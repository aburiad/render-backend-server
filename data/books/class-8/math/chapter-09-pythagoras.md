# নবম অধ্যায় — পিথাগোরাসের উপপাদ্য

## ভূমিকা

খ্রিষ্টপূর্ব ৬ষ্ঠ শতাব্দীর গ্রিক দার্শনিক পিথাগোরাস সমকোণী ত্রিভুজের একটি বিশেষ বৈশিষ্ট্য নিরূপণ করেন। সমকোণী ত্রিভুজের এ বৈশিষ্ট্য পিথাগোরাসের বৈশিষ্ট্য বলে পরিচিত। বলা হয় পিথাগোরাসের জন্মের অনেক পূর্ব থেকেই ব্যাবিলনীয়, চীনা ও মিশরীয়রা সমকোণী ত্রিভুজের এ বৈশিষ্ট্যের ব্যবহার ছিল। এ অধ্যায়ে আমরা সমকোণী ত্রিভুজের এ বৈশিষ্ট্য নিয়ে আলোচনা করব। সমকোণী ত্রিভুজের বাহুগুলো বিশেষ নামে পরিচিত। সমকোণের বিপরীত বাহুটি অতিভুজ $(Hypotenuse)$ এবং সমকোণ সংলগ্ন বাহুদ্বয় যথাক্রমে ভূমি $(Base)$ ও উচ্চতা $(Height)$। বর্তমান অধ্যায়ে এ তিনটি বাহুর দৈর্ঘ্যের মধ্যে যে সম্পর্ক রয়েছে, সে বিষয়ে আলোচনা করা হবে।

## অধ্যায় শেষে শিক্ষার্থীরা—

- পিথাগোরাসের উপপাদ্য যাচাই ও প্রমাণ করতে পারবে।
- ত্রিভুজের তিনটি বাহুর দৈর্ঘ্য দেওয়া থাকলে ত্রিভুজটি সমকোণী কি না যাচাই করতে পারবে।
- পিথাগোরাসের সূত্র ব্যবহার করে সমস্যা সমাধান করতে পারবে।

## ৯.১ সমকোণী ত্রিভুজ

চিত্রে, $ABC$ একটি সমকোণী ত্রিভুজ, যার $\angle ACB$ কোণটি সমকোণ। সুতরাং, $AB$ ত্রিভুজটির অতিভুজ। চিত্রে ত্রিভুজটির বাহুগুলো $a, b, c$ দ্বারা নির্দেশ করা হলো।

> **[চিত্র: সমকোণী ত্রিভুজ ABC যেখানে ∠C = 90°, বাহু a, b, c চিহ্নিত]**

### কাজ

1. একটি সমকোণ আঁকো এবং এর বাহু দুইটির উপর যথাক্রমে $3$ সে.মি. ও $4$ সে.মি. দূরত্বে দুইটি বিন্দু চিহ্নিত করো। বিন্দু দুইটি যোগ করে একটি সমকোণী ত্রিভুজ আঁকো। ত্রিভুজটির অতিভুজের দৈর্ঘ্য পরিমাপ করো। দেখো $5$ সে.মি. হয়েছে কি?

লক্ষ কর, $3^2 + 4^2 = 5^2$ অর্থাৎ দুই বাহুর দৈর্ঘ্য পরিমাপের বর্গের যোগফল অতিভুজের পরিমাপের বর্গের সমান।

সুতরাং, $a, b, c$ বাহু দ্বারা নির্দেশিত ত্রিভুজের ক্ষেত্রে $c^2 = a^2 + b^2$ হবে। এটি পিথাগোরাসের উপপাদ্যের মূল প্রতিপাদ্য। এই উপপাদ্যটি বিভিন্নভাবে প্রমাণ করা হয়েছে। এখানে কয়েকটি সহজ প্রমাণ দেওয়া হলো।

## ৯.২ পিথাগোরাসের উপপাদ্য

**একটি সমকোণী ত্রিভুজের অতিভুজের ওপর অঙ্কিত বর্গক্ষেত্রের অপর দুই বাহুর ওপর অঙ্কিত বর্গক্ষেত্রদ্বয়ের সমষ্টির সমান।**

*(দুইটি সমকোণী ত্রিভুজের সাহায্যে)*

**বিশেষ নির্বচন :** মনে করি, $ABC$ সমকোণী ত্রিভুজের $\angle B = 90°$ অতিভুজ $AC = b, AB = c$ ও $BC = a$।
প্রমাণ করতে হবে যে, $AC^2 = AB^2 + BC^2$
অর্থাৎ, $b^2 = c^2 + a^2$।

**অঙ্কন :** $BC$ কে $D$ পর্যন্ত বর্ধিত করি, যেন $CD = AB = c$ হয়।
$D$ বিন্দুতে বর্ধিত $BC$ এর উপর $DE$ লম্ব আঁকি, যেন $DE = BC = a$ হয়। $C, E$ ও $A, E$ যোগ করি।

> **[চিত্র: সমকোণী ত্রিভুজ ABC সঙ্গে CD = c, DE = a, ABDE ট্রাপিজিয়াম]**

**প্রমাণ :**

| ধাপ | যথার্থতা |
|---|---|
| (1) $\triangle ABC$ ও $\triangle CDE$ এ $AB = CD = c, BC = DE = a$ এবং অন্তর্ভুক্ত $\angle ABC = $ অন্তর্ভুক্ত $\angle CDE$ <br> সুতরাং, $\triangle ABC \cong \triangle CDE$ <br> $\therefore AC = CE = b$ এবং $\angle BAC = \angle ECD$ | [প্রত্যেকে সমকোণ] <br> [বাহু-কোণ-বাহু উপপাদ্য] |
| (2) আবার, $AB \perp BD$ এবং $ED \perp BD$ বলে $AB \parallel ED$ <br> সুতরাং, $ABDE$ একটি ট্রাপিজিয়াম। | |
| (3) তদুপরি, $\angle ACB + \angle BAC = \angle ACB + \angle ECD = $ এক সমকোণ। <br> $\therefore \angle ACE = $ এক সমকোণ। <br> এখন, $ABDE$ ট্রাপিজিয়ামক্ষেত্রের ক্ষেত্রফল <br> $= (\triangle$ ক্ষেত্র $ABC + \triangle$ ক্ষেত্র $CDE + \triangle$ ক্ষেত্র $ACE)$ | $\therefore \angle BAC = \angle ECD$ <br> $[\because \angle ACB + \angle ACE + \angle ECD = 2$ সমকোণ$]$ |
| বা, $\dfrac{1}{2}BD(AB + DE) = \dfrac{1}{2}ac + \dfrac{1}{2}ac + \dfrac{1}{2}b^2$ | [ট্রাপিজিয়াম ক্ষেত্রের ক্ষেত্রফল <br> $= \dfrac{1}{2}$ সমান্তরাল বাহুদ্বয়ের যোগফল × সমান্তরাল বাহুদ্বয়ের মধ্যবর্তী দূরত্ব] |
| বা, $\dfrac{1}{2}(BC + CD)(AB + DE) = \dfrac{1}{2}[2ac + b^2]$ | |
| বা, $(a + c)(a + c) = 2ac + b^2$ [2 দ্বারা গুণ করে] | |
| বা, $a^2 + 2ac + c^2 = 2ac + b^2$ | |
| $\therefore b^2 = c^2 + a^2$ (প্রমাণিত) | |

### পিথাগোরাসের উপপাদ্যের বিকল্প প্রমাণ (সদৃশকোণী ত্রিভুজের সাহায্যে)

**বিশেষ নির্বচন :** মনে করি, $ABC$ সমকোণী ত্রিভুজের $\angle C = 90°$ এবং অতিভুজ $AB = c, BC = a, AC = b$। প্রমাণ করতে হবে যে, $AB^2 = AC^2 + BC^2$,
অর্থাৎ $c^2 = a^2 + b^2$।

**অঙ্কন :** $C$ বিন্দু থেকে অতিভুজ $AB$ এর উপর লম্ব $CH$ অঙ্কন করি। $AB$ অতিভুজ $H$ বিন্দুতে $d$ ও $e$ অংশে বিভক্ত হলো।

> **[চিত্র: সমকোণী ত্রিভুজ ABC, AC = b, BC = a, AB = c, CH লম্ব, AH = d, BH = e]**

**প্রমাণ :**

| ধাপ | যথার্থতা |
|---|---|
| $\triangle BCH$ ও $\triangle ABC$ এ <br> $\angle BHC = \angle ACB$ এবং <br> $\angle CBH = \angle ABC$ | প্রত্যেকটি সমকোণ <br> সাধারণ কোণ |
| (1) $\therefore \triangle BCH$ ও $\triangle ABC$ সদৃশ। <br> $\therefore \dfrac{BC}{AB} = \dfrac{BH}{BC}$ <br> $\therefore \dfrac{a}{c} = \dfrac{e}{a} \quad ...(1)$ | [সদৃশ ত্রিভুজের অনুরূপ বাহুগুলো সমানুপাতিক] |
| (2) অনুরূপভাবে, $\triangle ACH$ ও $\triangle ABC$ সদৃশ। <br> $\therefore \dfrac{b}{c} = \dfrac{d}{b} \quad ...(2)$ | (i) উভয় ত্রিভুজ সমকোণী। <br> (ii) $\angle A$ কোণ সাধারণ। |
| (3) অনুরূপ দুইটি সূত্র থেকে পাই, <br> $a^2 = c \times e, b^2 = c \times d$ <br> অতএব, $a^2 + b^2 = c \times e + c \times d$ <br> $\quad\quad\quad\quad = c(e + d) = c \times c = c^2$ <br> $\therefore c^2 = a^2 + b^2$ (প্রমাণিত) | $\therefore c = e + d$ |

### পিথাগোরাসের উপপাদ্যের বিকল্প প্রমাণ (বীজগণিতের সাহায্যে)

পিথাগোরাসের উপপাদ্য বীজগণিতের সাহায্যে সহজেই প্রমাণ করা যায়।

**বিশেষ নির্বচন :** মনে করি, একটি সমকোণী ত্রিভুজের অতিভুজ $c$ এবং $a, b$ যথাক্রমে অন্য দুই বাহু। প্রমাণ করতে হবে, $c^2 = a^2 + b^2$।

**অঙ্কন :** প্রদত্ত ত্রিভুজটির সমান করে চারটি ত্রিভুজ চিত্রে প্রদর্শিত উপায়ে আঁকি।

> **[চিত্র: বড় বর্গক্ষেত্র (বাহু a+b) যার ভিতরে ছোট বর্গক্ষেত্র (বাহু c) এবং 4টি সমকোণী ত্রিভুজ]**

**প্রমাণ :**

| ধাপ | যথার্থতা |
|---|---|
| (1) অঙ্কিত বড় ক্ষেত্রটি বর্গক্ষেত্র। <br> এর ক্ষেত্রফল $(a + b)^2$ | [বাহুগুলোর প্রত্যেকটির দৈর্ঘ্য $a + b$ এবং কোণগুলো সমকোণ] |
| (2) ছোট চতুর্ভুজ ক্ষেত্রটি বর্গক্ষেত্রের। <br> এর ক্ষেত্রফল $c^2$ | [বাহুগুলোর প্রত্যেকটির দৈর্ঘ্য $c$] |
| (3) অঙ্কনানুসারে, বড় বর্গক্ষেত্রের ক্ষেত্রফল চারটি ত্রিভুজক্ষেত্র ও ছোট বর্গক্ষেত্রের ক্ষেত্রফলের সমষ্টির সমান। | |
| অর্থাৎ, $(a + b)^2 = 4 \times \dfrac{1}{2} \times a \times b + c^2$ | |
| বা, $a^2 + 2ab + b^2 = 2ab + c^2$ | |
| $\therefore c^2 = a^2 + b^2$ (প্রমাণিত) | |

### কাজ

1. $(a - b)^2$ এর বিকৃতির সাহায্যে পিথাগোরাসের উপপাদ্যটি প্রমাণ করো।

## ৯.৩ পিথাগোরাসের উপপাদ্যের বিপরীত উপপাদ্য

**যদি কোনো ত্রিভুজের একটি বাহুর উপর অঙ্কিত বর্গক্ষেত্র অপর দুই বাহুর উপর অঙ্কিত বর্গক্ষেত্রদ্বয়ের সমষ্টির সমান হয়, তবে শেষোক্ত বাহুদ্বয়ের অন্তর্ভুক্ত কোণটি সমকোণ হবে।**

**বিশেষ নির্বচন :** মনে করি, $\triangle ABC$ এর $AB^2 = AC^2 + BC^2$।
প্রমাণ করতে হবে যে, $\angle C = $ এক সমকোণ।

**অঙ্কন :** এমন একটি ত্রিভুজ $DEF$ আঁকি, যেন $\angle F$ এক সমকোণ, $EF = BC$ এবং $DF = AC$ হয়।

> **[চিত্র: ত্রিভুজ ABC এবং ত্রিভুজ DEF সমকোণী ∠F সহ]**

**প্রমাণ :**

| ধাপ | যথার্থতা |
|---|---|
| (1) $DE^2 = EF^2 + DF^2$ <br> $\quad\quad = BC^2 + AC^2 = AB^2$ <br> $\therefore DE = AB$ | [কারণ $\triangle DEF$ এ $\angle F$ এক সমকোণ] <br> [অঙ্কন অনুসারে] <br> [করণ] |
| এখন $\triangle ABC$ ও $\triangle DEF$ এ <br> $BC = EF, AC = DF$ এবং $AB = DE$। <br> $\therefore \triangle ABC \cong \triangle DEF$ <br> $\therefore \angle C = \angle F$ | [বাহু-বাহু-বাহু সর্বসমতা] |
| $\therefore \angle C = $ এক সমকোণ। (প্রমাণিত) | $[\because \angle F$ এক সমকোণ$]$ |
