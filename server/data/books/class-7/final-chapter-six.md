## অধ্যায় শেষে শিক্ষার্থীরা—

- বীজগণিতীয় ভগ্নাংশের লঘিষ্ঠ রূপ নির্ণয় করতে পারবে।
- সাধারণ হরবিশিষ্ট ভগ্নাংশে রূপান্তর করতে পারবে।
- বীজগণিতীয় ভগ্নাংশের যোগ, বিয়োগ ও সরলীকরণ করতে পারবে।

---

## ৬·১ বীজগণিতীয় ভগ্নাংশ

যে ভগ্নাংশের লব ও হর উভয়ই বীজগণিতীয় রাশি, তাকে **বীজগণিতীয় ভগ্নাংশ** বলা হয়।

যেমন: $\dfrac{a}{b}$, $\dfrac{x^2 + y}{2x}$, $\dfrac{3a + 5b}{a - b}$ ইত্যাদি বীজগণিতীয় ভগ্নাংশ।

---

## ৬·২ বীজগণিতীয় ভগ্নাংশের সরলীকরণ

বীজগণিতীয় ভগ্নাংশের লব ও হরকে সাধারণ উৎপাদক দ্বারা ভাগ করে যে রূপ পাওয়া যায়, তাকে ভগ্নাংশের **লঘিষ্ঠ রূপ** বলে।

> **নিয়ম:** ভগ্নাংশের লব ও হরকে একই রাশি দ্বারা গুণ বা ভাগ করলে ভগ্নাংশের মানের কোনো পরিবর্তন হয় না।

---

## ৬·৩ সাধারণ হরবিশিষ্ট ভগ্নাংশ

দুই বা ততোধিক ভগ্নাংশের হর সমান হলে, তাদেরকে **সাধারণ হরবিশিষ্ট ভগ্নাংশ** বলা হয়।

### সাধারণ হরবিশিষ্ট ভগ্নাংশে প্রকাশ করার নিয়ম

১. ভগ্নাংশগুলোর হরের লসাগু বের করতে হয়।
২. লসাগুকে প্রত্যেক ভগ্নাংশের হর দ্বারা ভাগ করে ভাগফল বের করতে হয়।
৩. প্রাপ্ত ভাগফল দ্বারা সংশ্লিষ্ট ভগ্নাংশের লব ও হরকে গুণ করতে হয়।

**উদাহরণ ১।** সাধারণ হরবিশিষ্ট ভগ্নাংশে প্রকাশ করো: $\dfrac{a}{4x}$ ও $\dfrac{b}{2x^2}$

**সমাধান:** হর $4x$ এবং $2x^2$ এর লসাগু $= 4x^2$

প্রথম ভগ্নাংশ: $\dfrac{a}{4x} = \dfrac{a \times x}{4x \times x} = \dfrac{ax}{4x^2}$

দ্বিতীয় ভগ্নাংশ: $\dfrac{b}{2x^2} = \dfrac{b \times 2}{2x^2 \times 2} = \dfrac{2b}{4x^2}$

নির্ণেয় সাধারণ হরবিশিষ্ট ভগ্নাংশ: $\dfrac{ax}{4x^2}$ ও $\dfrac{2b}{4x^2}$

**উদাহরণ ২।** সাধারণ হরবিশিষ্ট ভগ্নাংশে রূপান্তর করো: $\dfrac{2}{a^2 - 4}$ ও $\dfrac{5}{a^2 + 3a - 10}$

**সমাধান:** ১ম ভগ্নাংশের হর $= a^2 - 4 = (a + 2)(a - 2)$

২য় ভগ্নাংশের হর $= a^2 + 3a - 10 = a^2 + 5a - 2a - 10 = (a - 2)(a + 5)$

হর দুইটির লসাগু $= (a + 2)(a - 2)(a + 5)$

১ম ভগ্নাংশ: $\dfrac{2}{(a + 2)(a - 2)} = \dfrac{2(a + 5)}{(a + 2)(a - 2)(a + 5)} = \dfrac{2(a + 5)}{(a^2 - 4)(a + 5)}$

২য় ভগ্নাংশ: $\dfrac{5}{(a - 2)(a + 5)} = \dfrac{5(a + 2)}{(a - 2)(a + 5)(a + 2)} = \dfrac{5(a + 2)}{(a^2 - 4)(a + 5)}$

নির্ণেয় ভগ্নাংশ: $\dfrac{2(a + 5)}{(a^2 - 4)(a + 5)}$ ও $\dfrac{5(a + 2)}{(a^2 - 4)(a + 5)}$

**উদাহরণ ৩।** সাধারণ হরবিশিষ্ট ভগ্নাংশে পরিণত করো: $\dfrac{2}{x^2 + 3x}$, $\dfrac{3}{x^2 + 5x + 6}$, $\dfrac{3}{x^2 - x - 12}$

**সমাধান:** ১ম ভগ্নাংশের হর $= x^2 + 3x = x(x + 3)$

২য় ভগ্নাংশের হর $= x^2 + 5x + 6 = x^2 + 2x + 3x + 6 = (x + 2)(x + 3)$

৩য় ভগ্নাংশের হর $= x^2 - x - 12 = x^2 + 3x - 4x - 12 = (x + 3)(x - 4)$

হর তিনটির লসাগু $= x(x + 2)(x + 3)(x - 4)$

১ম ভগ্নাংশ: $\dfrac{2}{x(x + 3)} = \dfrac{2(x + 2)(x - 4)}{x(x + 2)(x + 3)(x - 4)}$

২য় ভগ্নাংশ: $\dfrac{3}{(x + 2)(x + 3)} = \dfrac{3x(x - 4)}{x(x + 2)(x + 3)(x - 4)}$

৩য় ভগ্নাংশ: $\dfrac{3}{(x + 3)(x - 4)} = \dfrac{3x(x + 2)}{x(x + 2)(x + 3)(x - 4)}$

> **কাজ:**
> ১। রাশি তিনটির লসাগু নির্ণয় করো: $x^2 + 3x$, $x^2 + 5x + 6$, $x^2 - x - 12$
> ২। সাধারণ হরবিশিষ্ট ভগ্নাংশে প্রকাশ করো: $\dfrac{a}{2x}$, $\dfrac{b}{4y}$

## ৬·৪ বীজগণিতীয় ভগ্নাংশের যোগ, বিয়োগ ও সরলীকরণ

### বীজগণিতীয় ভগ্নাংশের যোগ ও বিয়োগের নিয়ম

- ভগ্নাংশগুলোকে লঘিষ্ঠ সাধারণ হরবিশিষ্ট করতে হয়।
- **যোগফলের** হর = লঘিষ্ঠ সাধারণ হর; লব = রূপান্তরিত ভগ্নাংশগুলোর লবের যোগফল।
- **বিয়োগফলের** হর = লঘিষ্ঠ সাধারণ হর; লব = রূপান্তরিত ভগ্নাংশগুলোর লবের বিয়োগফল।

### বীজগণিতীয় ভগ্নাংশের যোগ

**উদাহরণ ৪।** যোগ করো: $\dfrac{a}{x} + \dfrac{b}{x}$

**সমাধান:** $\dfrac{a}{x} + \dfrac{b}{x} = \dfrac{a + b}{x}$

**উদাহরণ ৫।** যোগফল নির্ণয় করো: $\dfrac{a}{2x} + \dfrac{b}{3y}$

**সমাধান:** $2x$ ও $3y$ এর লসাগু $= 6xy$

$\dfrac{a}{2x} = \dfrac{a \times 3y}{2x \times 3y} = \dfrac{3ay}{6xy}$

$\dfrac{b}{3y} = \dfrac{b \times 2x}{3y \times 2x} = \dfrac{2bx}{6xy}$

$\therefore \dfrac{a}{2x} + \dfrac{b}{3y} = \dfrac{3ay + 2bx}{6xy}$

### বীজগণিতীয় ভগ্নাংশের বিয়োগ

**উদাহরণ ৬।** বিয়োগ করো: $\dfrac{a}{y}$ থেকে $\dfrac{b}{y}$

**সমাধান:** $\dfrac{a}{y} - \dfrac{b}{y} = \dfrac{a - b}{y}$

**উদাহরণ ৭।** বিয়োগ করো: $\dfrac{a}{3x}$ থেকে $\dfrac{b}{2y}$

**সমাধান:** $3x$ ও $2y$ এর লসাগু $= 6xy$

$\dfrac{a}{3x} = \dfrac{a \times 2y}{3x \times 2y} = \dfrac{2ay}{6xy}$

$\dfrac{b}{2y} = \dfrac{b \times 3x}{2y \times 3x} = \dfrac{3bx}{6xy}$

$\therefore \dfrac{a}{3x} - \dfrac{b}{2y} = \dfrac{2ay - 3bx}{6xy}$

**উদাহরণ ৮।** বিয়োগফল নির্ণয় করো: $\dfrac{a + 2}{a - 2}$ থেকে $1$

**সমাধান:** $\dfrac{a + 2}{a - 2} - 1 = \dfrac{a + 2}{a - 2} - \dfrac{a - 2}{a - 2} = \dfrac{(a + 2) - (a - 2)}{a - 2} = \dfrac{a + 2 - a + 2}{a - 2} = \dfrac{4}{a - 2}$

### বীজগণিতীয় ভগ্নাংশের সরলীকরণ

প্রক্রিয়া চিহ্ন ($+$, $-$) দ্বারা সংযুক্ত দুই বা ততোধিক বীজগণিতীয় ভগ্নাংশকে একটি ভগ্নাংশে বা রাশিতে পরিণত করাই হলো **ভগ্নাংশের সরলীকরণ**। এতে প্রাপ্ত ভগ্নাংশটিকে লঘিষ্ঠ আকারে প্রকাশ করা হয়।

**উদাহরণ ৯।** সরল করো: $\dfrac{a}{a + b} + \dfrac{b}{a - b}$

**সমাধান:** $\dfrac{a}{a + b} + \dfrac{b}{a - b} = \dfrac{a(a - b) + b(a + b)}{(a + b)(a - b)} = \dfrac{a^2 - ab + ab + b^2}{a^2 - b^2} = \dfrac{a^2 + b^2}{a^2 - b^2}$

**উদাহরণ ১০।** সরল করো: $\dfrac{x + 1}{y + z} - \dfrac{y + z}{x + 1}$

**সমাধান:** $\dfrac{x + 1}{y + z} - \dfrac{y + z}{x + 1} = \dfrac{(x + 1)(x + 1) - (y + z)(y + z)}{(y + z)(x + 1)} = \dfrac{(x + 1)^2 - (y + z)^2}{(y + z)(x + 1)}$

**উদাহরণ ১১।** সরল করো: $\dfrac{x - 1}{x + y} - \dfrac{y - z}{z + x} + \dfrac{x - y}{y - z}$ *[verify]*
