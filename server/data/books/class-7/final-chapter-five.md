# পঞ্চম অধ্যায় — বীজগণিতীয় সূত্রাবলি ও প্রয়োগ

## ভূমিকা

বীজগণিতীয় প্রতীক দ্বারা প্রকাশিত যেকোনো সাধারণ নিয়ম বা সিদ্ধান্তকে **বীজগণিতীয় সূত্র** বা সংক্ষেপে **সূত্র** বলা হয়। আমরা বিভিন্ন ক্ষেত্রে সূত্র ব্যবহার করে থাকি। এ অধ্যায়ে প্রথম চারটি সূত্র এবং এ চারটি সূত্রের সাহায্যে অনুসিদ্ধান্ত নির্ণয়ের পদ্ধতি দেখানো হয়েছে। এ ছাড়া বীজগণিতীয় সূত্র ও অনুসিদ্ধান্ত প্রয়োগ করে বীজগণিতীয় রাশির মান নির্ণয় ও উৎপাদকে বিশ্লেষণ উপস্থাপন করা হয়েছে। আবার বীজগণিতীয় রাশির সাহায্যে ভাজ্য, ভাজক, গুণনীয়ক, গুণিতক সম্পর্কে ধারণা দেওয়া হয়েছে এবং কীভাবে অনূর্ধ্ব তিনটি বীজগণিতীয় রাশির গসাগু ও লসাগু নির্ণয় করা যায় তা আলোচনা করা হয়েছে।

## অধ্যায় শেষে শিক্ষার্থীরা—

- বর্গ নির্ণয়ে বীজগণিতীয় সূত্রের বর্ণনা ও প্রয়োগ করতে পারবে।
- বীজগণিতীয় সূত্র ও অনুসিদ্ধান্ত প্রয়োগ করে রাশির মান নির্ণয় করতে পারবে।
- বীজগণিতীয় সূত্র প্রয়োগ করে উৎপাদকে বিশ্লেষণ করতে পারবে।
- গুণনীয়ক ও গুণিতক কী তা ব্যাখ্যা করতে পারবে।
- অনূর্ধ্ব তিনটি বীজগণিতীয় রাশির সাংখ্যিক সহগসহ গসাগু ও লসাগু নির্ণয় করতে পারবে।

## ৫·১ বীজগণিতীয় সূত্রাবলি

### সূত্র ১

$(a + b)^2 = (a + b)(a + b) = a(a + b) + b(a + b) = a^2 + ab + ba + b^2 = a^2 + 2ab + b^2$

> **রাশির যোগফলের বর্গ = ১ম রাশির বর্গ + ২ × ১ম রাশি × ২য় রাশি + ২য় রাশির বর্গ**

### সূত্রটির জ্যামিতিক ব্যাখ্যা

$ABCD$ একটি বর্গক্ষেত্র যার $AB$ বাহু $= a + b$, $BC$ বাহু $= a + b$।

$ABCD$ বর্গক্ষেত্রের ক্ষেত্রফল $= (\text{বাহুর দৈর্ঘ্য})^2 = (a + b)^2$

বর্গক্ষেত্রটিকে $P$, $Q$, $R$, $S$ চারটি ভাগে ভাগ করা হয়েছে। এখানে $P$ ও $S$ বর্গক্ষেত্র এবং $Q$ ও $R$ আয়তক্ষেত্র।

| অংশ | ক্ষেত্রফল |
|---|---|
| $P$ | $a \times a = a^2$ |
| $Q$ | $a \times b = ab$ |
| $R$ | $b \times a = ab$ |
| $S$ | $b \times b = b^2$ |

$\therefore (a + b)^2 = a^2 + ab + ab + b^2 = a^2 + 2ab + b^2$

### অনুসিদ্ধান্ত ১

আমরা জানি, $(a + b)^2 = a^2 + 2ab + b^2$

বা, $a^2 + b^2 = (a + b)^2 - 2ab$ [উভয়পক্ষ থেকে $2ab$ বিয়োগ করে]

**উদাহরণ ১।** $(m + n)$ এর বর্গ নির্ণয় কর।

**সমাধান:** $(m + n)^2 = (m)^2 + 2 \times m \times n + (n)^2 = m^2 + 2mn + n^2$

**উদাহরণ ২।** $(3x + 4)$ এর বর্গ নির্ণয় কর।

**সমাধান:** $(3x + 4)^2 = (3x)^2 + 2 \times 3x \times 4 + (4)^2 = 9x^2 + 24x + 16$

**উদাহরণ ৩।** $(2x + 3y)$ এর বর্গ নির্ণয় করো।

**সমাধান:** $(2x + 3y)^2 = (2x)^2 + 2 \times 2x \times 3y + (3y)^2 = 4x^2 + 12xy + 9y^2$

**উদাহরণ ৪।** বর্গের সূত্র প্রয়োগ করে $105$ এর বর্গ নির্ণয় করো।

**সমাধান:** $(105)^2 = (100 + 5)^2 = (100)^2 + 2 \times 100 \times 5 + (5)^2 = 10000 + 1000 + 25 = 11025$

> **কাজ:** সূত্রের সাহায্যে রাশিগুলোর বর্গ নির্ণয় করো:
> ১। $y + 2z$ ২। $3a + 5b$ ৩। $5p + 2q$ ৪। $15$ ৫। $103$

### সূত্র ২

$(a - b)^2 = (a - b)(a - b) = a^2 - ab - ba + b^2 = a^2 - 2ab + b^2$

> **রাশির বিয়োগফলের বর্গ = ১ম রাশির বর্গ - ২ × ১ম রাশি × ২য় রাশি + ২য় রাশির বর্গ**

দ্বিতীয় সূত্রটি প্রথম সূত্রের সাহায্যেও নির্ণয় করা যায়:

$(a - b)^2 = \{a + (-b)\}^2 = a^2 + 2 \times a \times (-b) + (-b)^2 = a^2 - 2ab + b^2$ [$b$ এর পরিবর্তে $-b$ বসিয়ে]

### অনুসিদ্ধান্ত ২

$a^2 + b^2 = (a - b)^2 + 2ab$ [উভয়পক্ষে $2ab$ যোগ করে]

**উদাহরণ ৫।** $(p - q)$ এর বর্গ নির্ণয় করো।

**সমাধান:** $(p - q)^2 = (p)^2 - 2 \times p \times q + (q)^2 = p^2 - 2pq + q^2$

**উদাহরণ ৬।** $(5x - 3y)$ এর বর্গ নির্ণয় করো।

**সমাধান:** $(5x - 3y)^2 = (5x)^2 - 2 \times 5x \times 3y + (3y)^2 = 25x^2 - 30xy + 9y^2$

**উদাহরণ ৭।** বর্গের সূত্র প্রয়োগ করে $98$ এর বর্গ নির্ণয় করো।

**সমাধান:** $(98)^2 = (100 - 2)^2 = (100)^2 - 2 \times 100 \times 2 + (2)^2 = 10000 - 400 + 4 = 9604$

> **কাজ:** সূত্রের সাহায্যে রাশিগুলোর বর্গ নির্ণয় করো:
> ১। $5x - 3$ ২। $ax - by$ ৩। $7x - 6$ ৪। $95$

### প্রথম ও দ্বিতীয় সূত্রের আরও কয়েকটি অনুসিদ্ধান্ত

**অনুসিদ্ধান্ত ৩:** $a^2 + b^2 = (a + b)^2 - 2ab$

**অনুসিদ্ধান্ত ৪:** $(a + b)^2 = (a - b)^2 + 4ab$

**অনুসিদ্ধান্ত ৫:** $(a - b)^2 = (a + b)^2 - 4ab$

**অনুসিদ্ধান্ত ৬:** $(a + b)^2 + (a - b)^2 = 2(a^2 + b^2)$

**অনুসিদ্ধান্ত ৭:** $(a + b)^2 - (a - b)^2 = 4ab$

**উদাহরণ ৮।** $a + b = 7$ এবং $ab = 9$ হলে, $a^2 + b^2$ এর মান নির্ণয় করো।

**সমাধান:** আমরা জানি, $a^2 + b^2 = (a + b)^2 - 2ab = (7)^2 - 2 \times 9 = 49 - 18 = 31$

**উদাহরণ ৯।** $a + b = 5$ এবং $ab = 6$ হলে, $(a - b)^2$ এর মান নির্ণয় করো।

**সমাধান:** আমরা জানি, $(a - b)^2 = (a + b)^2 - 4ab = (5)^2 - 4 \times 6 = 25 - 24 = 1$

**উদাহরণ ১০।** $p - \dfrac{1}{p} = 8$ হলে, প্রমাণ করো যে, $p^2 + \dfrac{1}{p^2} = 66$

**সমাধান:** $p^2 + \dfrac{1}{p^2} = \left(p - \dfrac{1}{p}\right)^2 + 2 = (8)^2 + 2 = 64 + 2 = 66$ (প্রমাণিত)

**বিকল্প পদ্ধতি:**

$\left(p - \dfrac{1}{p}\right)^2 = (8)^2$ [উভয়পক্ষকে বর্গ করে]

বা, $p^2 - 2 \times p \times \dfrac{1}{p} + \dfrac{1}{p^2} = 64$

বা, $p^2 - 2 + \dfrac{1}{p^2} = 64$

বা, $p^2 + \dfrac{1}{p^2} = 64 + 2 = 66$ (প্রমাণিত)

> **কাজ:**
> ১। $a + b = 4$ এবং $ab = 2$ হলে, $(a - b)^2$ এর মান নির্ণয় কর।
> ২। $a - \dfrac{1}{a} = 5$ হলে, দেখাও যে, $a^2 + \dfrac{1}{a^2} = 27$

---

**উদাহরণ ১১।** $(a + b + c)$ এর বর্গ নির্ণয় করো।

**সমাধান:** ধরি, $a + b = p$

$(a + b + c)^2 = \{(a + b) + c\}^2 = (p + c)^2 = p^2 + 2pc + c^2$

[$p$ এর মান বসিয়ে] $= (a + b)^2 + 2(a + b)c + c^2$

$= a^2 + 2ab + b^2 + 2ac + 2bc + c^2$

$= a^2 + b^2 + c^2 + 2ab + 2bc + 2ca$

**উদাহরণ ১২।** $(x + y - z)$ এর বর্গ নির্ণয় করো।

**সমাধান:** ধরি, $x + y = m$

$(x + y - z)^2 = (m - z)^2 = m^2 - 2mz + z^2$

[$m$-এর মান বসিয়ে] $= (x + y)^2 - 2(x + y)z + z^2$

$= x^2 + 2xy + y^2 - 2xz - 2yz + z^2$

$= x^2 + y^2 + z^2 + 2xy - 2xz - 2yz$

**উদাহরণ ১৩।** $(3x - 2y + 5z)$ এর বর্গ নির্ণয় করো।

**সমাধান:** $(3x - 2y + 5z)^2 = \{(3x - 2y) + 5z\}^2$

$= (3x - 2y)^2 + 2(3x - 2y)(5z) + (5z)^2$

$= 9x^2 - 12xy + 4y^2 + 30xz - 20yz + 25z^2$

$= 9x^2 + 4y^2 + 25z^2 - 12xy + 30xz - 20yz$

**উদাহরণ ১৪।** সরল করো: $(2x + 3y)^2 - 2(2x + 3y)(2x - 5y) + (2x - 5y)^2$

**সমাধান:** ধরি, $2x + 3y = a$ এবং $2x - 5y = b$

প্রদত্ত রাশি $= a^2 - 2ab + b^2 = (a - b)^2$

$= \{(2x + 3y) - (2x - 5y)\}^2 = (8y)^2 = 64y^2$

**উদাহরণ ১৫।** $x = 7$ এবং $y = 6$ হলে, $16x^2 - 40xy + 25y^2$ এর মান নির্ণয় করো।

**সমাধান:** $16x^2 - 40xy + 25y^2 = (4x)^2 - 2 \times 4x \times 5y + (5y)^2 = (4x - 5y)^2$

$= (4 \times 7 - 5 \times 6)^2 = (28 - 30)^2 = (-2)^2 = 4$

> **কাজ:**
> ১। $(3x - 2y - z)$ এর বর্গ নির্ণয় করো।
> ২। সরল কর: $(5a - 7b)^2 + 2(5a - 7b)(9b - 4a) + (9b - 4a)^2$
> ৩। $x = 3$ হলে, $9x^2 - 24x + 16$ এর মান কত?

### সূত্র ৩

$(a + b)(a - b) = a^2 - b^2$

**প্রমাণ:** $(a + b)(a - b) = a(a - b) + b(a - b) = a^2 - ab + ab - b^2 = a^2 - b^2$

> **রাশির যোগফল × রাশির বিয়োগফল = রাশি দুটির বর্গের বিয়োগফল**

**উদাহরণ ১৬।** সূত্রের সাহায্যে $(3x + 2y)$ কে $(3x - 2y)$ দ্বারা গুণ করো।

**সমাধান:** $(3x + 2y)(3x - 2y) = (3x)^2 - (2y)^2 = 9x^2 - 4y^2$

**উদাহরণ ১৭।** সূত্রের সাহায্যে $(ax^2 + b)$ কে $(ax^2 - b)$ দ্বারা গুণ করো।

**সমাধান:** $(ax^2 + b)(ax^2 - b) = (ax^2)^2 - (b)^2 = a^2x^4 - b^2$

**উদাহরণ ১৮।** সূত্রের সাহায্যে $(3x + 2y + 1)$ কে $(3x - 2y + 1)$ দ্বারা গুণ করো।

**সমাধান:** $(3x + 2y + 1)(3x - 2y + 1) = \{(3x + 1) + 2y\}\{(3x + 1) - 2y\}$

$= (3x + 1)^2 - (2y)^2 = 9x^2 + 6x + 1 - 4y^2$

$= 9x^2 - 4y^2 + 6x + 1$

### সূত্র ৪

$(x + a)(x + b) = x^2 + (a + b)x + ab$

**প্রমাণ:** $(x + a)(x + b) = x^2 + bx + ax + ab = x^2 + (a + b)x + ab$

> অর্থাৎ, $(x + a)(x + b) = x^2 + (a \text{ ও } b \text{ এর যোগফল}) \times x + (a \text{ ও } b \text{ এর গুণফল})$

**উদাহরণ ১৯।** $(a + 3)$ কে $(a + 2)$ দ্বারা গুণ করো।

**সমাধান:** $(a + 3)(a + 2) = a^2 + (3 + 2)a + 3 \times 2 = a^2 + 5a + 6$

**উদাহরণ ২০।** $(px + 3)$ কে $(px - 5)$ দ্বারা গুণ করো।

**সমাধান:** $(px + 3)(px - 5) = (px)^2 + \{3 + (-5)\}px + 3 \times (-5)$

$= p^2x^2 + (-2)px - 15 = p^2x^2 - 2px - 15$

**উদাহরণ ২১।** $(p^2 - 2r)$ কে $(p^2 - 3r)$ দ্বারা গুণ করো।

**সমাধান:** $(p^2 - 2r)(p^2 - 3r) = (p^2)^2 + (-2r - 3r)p^2 + (-2r)(-3r)$

$= p^4 - 5p^2r + 6r^2$

**উদাহরণ ২২।** সূত্রের সাহায্যে গুণফল নির্ণয় করো: $(2x + y)$, $(2x - y)$, $(4x^2 + y^2)$

**সমাধান:** $(2x + y)(2x - y)(4x^2 + y^2) = \{(2x)^2 - y^2\}(4x^2 + y^2)$

$= (4x^2 - y^2)(4x^2 + y^2) = (4x^2)^2 - (y^2)^2 = 16x^4 - y^4$

> **কাজ:**
> ১। $(2a + 3)(2a - 3)$ দ্বারা গুণ করো।
> ২। $(4x + 5)$ কে $(4x + 3)$ দ্বারা গুণ করো।
> ৩। $(6a - 7)(6a + 5)$ দ্বারা গুণ করো।

## ৫·২ বীজগণিতীয় রাশির উৎপাদক

আমরা জানি, $6 = 2 \times 3$। এখানে $2$ ও $3$ হলো $6$ এর দুটি উৎপাদক বা গুণনীয়ক।

৩ নং সূত্র থেকে আমরা জানি, $a^2 - b^2 = (a + b)(a - b)$

তাহলে, $(a + b)$ ও $(a - b)$ বীজগণিতীয় রাশি $a^2 - b^2$ এর দুটি উৎপাদক বা গুণনীয়ক।

> কোনো বীজগণিতীয় রাশি দুই বা ততোধিক রাশির গুণফল হলে, শেষোক্ত রাশিগুলোর প্রত্যেকটিকে প্রথম রাশির **উৎপাদক** বা **গুণনীয়ক** বলা হয়।

বীজগণিতীয় বিভিন্ন সূত্র এবং গুণের বিনিময়বিধি, সংযোগবিধি ও বণ্টনবিধি ব্যবহার করে বীজগণিতীয় রাশিকে উৎপাদকে বিশ্লেষণ করা হয়।

### গুণের বণ্টনবিধির সাহায্যে উৎপাদকে বিশ্লেষণ

**উদাহরণ ২৩।** $20x + 4y$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $20x + 4y = 4 \times 5x + 4 \times y = 4(5x + y)$ [গুণের বণ্টনবিধি অনুযায়ী]

**উদাহরণ ২৪।** $ax - by - ay + by$ কে উৎপাদকে বিশ্লেষণ করো। *[verify]*

**উদাহরণ ২৫।** উৎপাদকে বিশ্লেষণ করো: $2x - 6x^2$

**সমাধান:** $2x - 6x^2 = 2x(1 - 3x)$

**উদাহরণ ২৬।** উৎপাদকে বিশ্লেষণ করো: $x^2 + 4x + xy + 4y$

**সমাধান:** $x^2 + 4x + xy + 4y = x(x + 4) + y(x + 4) = (x + 4)(x + y)$

> **লক্ষ করি:** দুটি রাশি এমনভাবে নির্বাচন করতে হবে যেন বণ্টনবিধি প্রয়োগ করে প্রাপ্ত রাশি দুটির মধ্যে একটি সাধারণ উৎপাদক পাওয়া যায়।

### বীজগণিতীয় সূত্রের সাহায্যে উৎপাদকে বিশ্লেষণ

**উদাহরণ ২৭।** উৎপাদকে বিশ্লেষণ করো: $25 - 9x^2$

**সমাধান:** $25 - 9x^2 = (5)^2 - (3x)^2 = (5 + 3x)(5 - 3x)$

**উদাহরণ ২৮।** $8a^4 - 2x^2$ কে উৎপাদকে বিশ্লেষণ কর। *[verify]*

**উদাহরণ ২৯।** উৎপাদকে বিশ্লেষণ করো: $25(a + 2b)^2 - 36(2a - 5b)^2$

**সমাধান:** ধরি, প্রদত্ত রাশি $= 25x^2 - 36y^2 = (5x)^2 - (6y)^2 = (5x + 6y)(5x - 6y)$

$x$ ও $y$ এর মান বসিয়ে:

$= \{5(a + 2b) + 6(2a - 5b)\}\{5(a + 2b) - 6(2a - 5b)\}$

$= (5a + 10b + 12a - 30b)(5a + 10b - 12a + 30b)$

$= (17a - 20b)(40b - 7a)$

**উদাহরণ ৩০।** উৎপাদকে বিশ্লেষণ করো: $x^2 + 5x + 6$

**সমাধান:** $x^2 + 5x + 6 = x^2 + (2 + 3)x + 2 \times 3 = (x + 2)(x + 3)$

এখানে, $a = 2$ এবং $b = 3$

**উদাহরণ ৩১।** উৎপাদকে বিশ্লেষণ করো: $4x^2 - 4xy + y^2 - z^2$

**সমাধান:** $4x^2 - 4xy + y^2 - z^2 = (2x)^2 - 2 \times 2x \times y + (y)^2 - z^2$

$= (2x - y)^2 - z^2 = (2x - y + z)(2x - y - z)$

> **কাজ:** উৎপাদকে বিশ্লেষণ করো:
> ১। $a^2 - 81b^2$ ২। $25x^4 - 36y^4$ ৩। $9x^2 - (2x + y)^2$ ৪। $x^2 + 7x + 10$

## ৫·৩ ভাজ্য, ভাজক, গুণনীয়ক ও গুণিতক

$x$, $y$ ও $z$ তিনটি রাশি ধরি, $x \div y = z$

| | রাশি |
|---|---|
| ভাজ্য | $x$ |
| ভাজক | $y$ |
| ভাগফল | $z$ |

যেমন, $10 \div 2 = 5$

এখানে, $10$ ভাজ্য, $2$ ভাজক, $5$ ভাগফল।

এক্ষেত্রে $10$, $2$ এর একটি গুণিতক। আবার $10$, $5$ এরও একটি গুণিতক। অপরদিকে $2$ এবং $5$ উভয় $10$ এর উৎপাদক।

> একটি রাশি (ভাজ্য) অপর একটি রাশি (ভাজক) দ্বারা নিঃশেষে বিভাজ্য হলে, ভাজ্যকে ভাজকের একটি **গুণিতক** (Multiple) বলা হয় এবং ভাজককে ভাজ্যের **গুণনীয়ক** বা **উৎপাদক** (Factor) বলে।

## ৫·৪ গরিষ্ঠ সাধারণ গুণনীয়ক (গসাগু)

পাটিগণিত থেকে আমরা জেনেছি:

- $12$ এর গুণনীয়কগুলো: $1, 2, 3, 4, 6, 12$
- $18$ এর গুণনীয়কগুলো: $1, 2, 3, 6, 9, 18$
- $24$ এর গুণনীয়কগুলো: $1, 2, 3, 4, 6, 8, 12, 24$

$12, 18$ ও $24$ এর সাধারণ গুণনীয়কগুলো: $2, 3$ ও $6$। এদের মধ্যে বড় গুণনীয়কটি $6$।

$\therefore 12, 18$ ও $24$ এর গসাগু $= 6$

> দুই বা ততোধিক রাশির **গরিষ্ঠ সাধারণ গুণনীয়ক (গসাগু)** হলো এমন একটি রাশি যা সাধারণ গুণনীয়কগুলোর মধ্যে সবচেয়ে বড় মানের একটি রাশি এবং যা দ্বারা প্রদত্ত রাশিগুলো নিঃশেষে বিভাজ্য হয়।

### গসাগু নির্ণয়ের নিয়ম

১. পাটিগণিতের নিয়মে প্রদত্ত রাশিগুলোর সাংখ্যিক সহগের গসাগু নির্ণয় করতে হয়।
২. বীজগণিতীয় রাশিগুলোর মৌলিক উৎপাদক বের করতে হয়।
৩. সাংখ্যিক সহগের গসাগু এবং প্রদত্ত রাশিগুলোর বীজগণিতীয় সাধারণ মৌলিক উৎপাদকগুলোর ধারাবাহিক গুণফল হচ্ছে নির্ণেয় গসাগু।

**উদাহরণ ৩২।** $8x^2y^2$ এবং $10x^3y^2 - 3$ এর গসাগু নির্ণয় করো। *[verify]*

**উদাহরণ ৩৩।** $2(a^2 - b^2)$ এবং $(a^2 - 2ab + b^2)$ এর গসাগু নির্ণয় করো।

**সমাধান:** ১ম রাশি $= 2(a^2 - b^2) = 2(a + b)(a - b)$

২য় রাশি $= a^2 - 2ab + b^2 = (a - b)(a - b) = (a - b)^2$

এখানে সাংখ্যিক সহগ $2$ ও $1$ এর গসাগু $= 1$।

সাধারণ মৌলিক উৎপাদক $= (a - b)$

নির্ণেয় গসাগু $= 1 \times (a - b) = (a - b)$

**উদাহরণ ৩৪।** $x^2 - 4$, $2x + 4$ এবং $x^2 + 5x + 6$ এর গসাগু নির্ণয় করো।

**সমাধান:** ১ম রাশি $= x^2 - 4 = (x + 2)(x - 2)$

২য় রাশি $= 2x + 4 = 2(x + 2)$

৩য় রাশি $= x^2 + 5x + 6 = x^2 + 2x + 3x + 6 = x(x + 2) + 3(x + 2) = (x + 2)(x + 3)$

এখানে প্রদত্ত রাশিগুলোর সাংখ্যিক সহগ $1, 2$ এবং $1$ এর গসাগু $= 1$।

সাধারণ মৌলিক উৎপাদক $= (x + 2)$

নির্ণেয় গসাগু $= 1 \times (x + 2) = (x + 2)$

> **কাজ:** গসাগু নির্ণয় করো:
> ১। $3x^3y^2$, $2x^2y^3$
> ২। $3xy$, $9xy^2$
> ৩। $(x^2 - 25)$, $(x - 5)^2$
> ৪। $x^2 - 9$, $x^2 + 7x + 12$, $3x + 9$

## ৫·৫ লঘিষ্ঠ সাধারণ গুণিতক (লসাগু)

পাটিগণিতে আমরা জানি:

- $4$ এর গুণিতকগুলো: $4, 8, 12, 16, 20, 24, 28, 32, 36, \ldots$
- $6$ এর গুণিতকগুলো: $6, 12, 18, 24, 30, 36, \ldots$

$4$ এবং $6$ এর সাধারণ গুণিতক: $12, 24, 36, \ldots$

$4$ এবং $6$ এর লঘিষ্ঠ সাধারণ গুণিতক $= 12$।

> দুই বা ততোধিক রাশির সম্ভাব্য সকল উৎপাদকের সর্বোচ্চ ঘাতের গুণফলকে রাশিগুলোর **লঘিষ্ঠ সাধারণ গুণিতক (লসাগু)** বলা হয়।

### লসাগু নির্ণয়ের নিয়ম

১. লসাগু নির্ণয় করার জন্য প্রথমে সাংখ্যিক সহগগুলোর লসাগু বের করতে হবে।
২. এরপর উৎপাদকের সর্বোচ্চ ঘাত বের করতে হবে।
৩. অতঃপর উভয়ের গুণফলই হবে প্রদত্ত রাশিগুলোর লসাগু।

**উদাহরণ ৩৫।** $4x^2y$, $6xy^2$ এবং $9xyz$ এর লসাগু নির্ণয় করো। *[verify]*

**সমাধান:** রাশিগুলোর সাংখ্যিক সহগ $4, 6$ ও $9$ এর লসাগু $= 36$
