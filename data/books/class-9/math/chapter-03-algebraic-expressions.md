# তৃতীয় অধ্যায় — বীজগাণিতিক রাশি (Algebraic Expressions)

## ভূমিকা

বীজগণিতে অনেক সমস্যা সমাধানে বীজগাণিতিক সূত্র ব্যবহৃত হয়। আবার অনেক বীজগাণিতিক রাশি বিশ্লেষণ করে উৎপাদকের মাধ্যমে উপস্থাপন করা হয়ে থাকে। তাই এ অধ্যায়ে বীজগাণিতিক সূত্রের সাহায্যে সমস্যা সমাধান এবং রাশির উৎপাদকে বিশ্লেষণ বিষয়ক বিষয়বস্তু শিক্ষার্থীর উপযোগী করে উপস্থাপন করা হয়েছে। অধিকন্তু নানাবিধ গাণিতিক সমস্যা বীজগাণিতিক সূত্রের সাহায্যে উৎপাদকে বিশ্লেষণ করেও সমাধান করা যায়। পূর্বের শ্রেণিতে বীজগাণিতিক সূত্রাবলি ও এদের সাথে সম্পৃক্ত অনুসিদ্ধান্তসহ সংক্ষেপে বিস্তৃতির আলোচনা করা হয়েছে। এ অধ্যায়ে ঐগুলো পুনরুল্লেখ করা হলো এবং উদাহরণের মাধ্যমে এদের কতিপয় প্রয়োগ দেখানো হলো। এছাড়াও এ অধ্যায়ে বর্গ ও ঘনের সম্প্রসারণ, ভাগশেষ উপপাদ্য প্রয়োগে রাশির উৎপাদকে বিশ্লেষণ এবং বাস্তব সমস্যা সমাধানে বীজগাণিতিক সূত্রের গঠন ও প্রয়োগ সম্পর্কে বিস্তারিত আলোচনা করা হয়েছে।

## অধ্যায় শেষে শিক্ষার্থীরা—

- বীজগাণিতিক সূত্র প্রয়োগ করে বর্গ ও ঘন রাশির সম্প্রসারণ করতে পারবে;
- ভাগশেষ উপপাদ্য কী ব্যাখ্যা করতে পারবে এবং তা প্রয়োগে করে উৎপাদকে বিশ্লেষণ করতে পারবে;
- বাস্তব সমস্যা সমাধানের জন্য বীজগাণিতিক সূত্র গঠন করতে পারবে এবং সূত্র প্রয়োগ করে সমস্যা সমাধান করতে পারবে।

## ৩.১ বীজগাণিতিক রাশি

সংখ্যা নির্দেশক প্রতীক এবং প্রক্রিয়া চিহ্ন এর অর্থবোধক বিন্যাসকে **বীজগাণিতিক রাশি** বলা হয়। যেমন, $2a + 3b - 4c$ একটি বীজগাণিতিক রাশি। বীজগাণিতিক রাশিতে $a, b, c, p, q, r, m, n, x, y, z, \ldots$ ইত্যাদির বর্ণের মাধ্যমে বিভিন্ন তথ্য প্রকাশ করা হয়। বীজগণিতিকে রাশি সংবলিত বিভিন্ন সমস্যা সমাধানে এর সমষ্ট পর্যন্ত ব্যবহার করা হয়। পাটিগণিতে শুধু ধনাত্মক সংখ্যা ব্যবহার হয়, অন্যদিকে বীজগণিতে শূন্যসহ ধনাত্মক ও ঋণাত্মক সকল সংখ্যা ব্যবহার করা হয়। বীজগণিতকে পাটিগণিতের সর্বানুকৃত (generalized) রূপ বলা হয়। বীজগাণিতিক রাশিতে ব্যবহৃত সংখ্যা ও সংখ্যা প্রতীকগুলো **ধ্রুবক** (constant), এদের মান নির্দিষ্ট। আর অক্ষর প্রতীকগুলো **চলক** (variables), এদের মান নির্দিষ্ট নয়, এরা বিভিন্ন মান ধারণ করতে পারে।

## ৩.২ বর্গ সংবলিত সূত্রাবলি

বীজগাণিতিক প্রতীক দ্বারা প্রকাশিত যেকোনো সাধারণ নিয়ম বা সিদ্ধান্তকে **বীজগাণিতিক সূত্র** বলা হয়। সপ্তম ও অষ্টম শ্রেণিতে বীজগাণিতিক সূত্রাবলি ও এতদসংক্রান্ত অনুসিদ্ধান্তগুলো সংক্ষেপে আলোচনা করা হয়েছে। এ অধ্যায়ে ঐগুলো পুনরুল্লেখ করে কতিপয় প্রয়োগ দেখানো হলো।

**সূত্র ১.** $\quad (a+b)^2 = a^2 + 2ab + b^2$

**সূত্র ২.** $\quad (a-b)^2 = a^2 - 2ab + b^2$

**মন্তব্য:** সূত্র ১ ও সূত্র ২ হতে দেখা যায় যে, $a^2 + b^2$ এর সাথে $2ab$ অথবা $-2ab$ যোগ করলে একটি পূর্ণবর্গ, অর্থাৎ $(a+b)^2$ অথবা $(a-b)^2$ পাওয়া যায়। সূত্র ১ এ $b$ এর স্থলে $-b$ বসালে সূত্র ২ পাওয়া যাবে; $\{a + (-b)\}^2 = a^2 + 2a(-b) + (-b)^2$ অর্থাৎ, $(a-b)^2 = a^2 - 2ab + b^2$

**অনুসিদ্ধান্ত ১.** $\quad a^2 + b^2 = (a+b)^2 - 2ab$

**অনুসিদ্ধান্ত ২.** $\quad a^2 + b^2 = (a-b)^2 + 2ab$

**অনুসিদ্ধান্ত ৩.** $\quad (a+b)^2 = (a-b)^2 + 4ab$

**প্রমাণ:** $(a+b)^2 = a^2 + 2ab + b^2 = a^2 - 2ab + b^2 + 4ab = (a-b)^2 + 4ab$

**অনুসিদ্ধান্ত ৪.** $\quad (a-b)^2 = (a+b)^2 - 4ab$

**প্রমাণ:** $(a-b)^2 = a^2 - 2ab + b^2 = a^2 + 2ab + b^2 - 4ab = (a+b)^2 - 4ab$

**অনুসিদ্ধান্ত ৫.** $\quad a^2 + b^2 = \dfrac{(a+b)^2 + (a-b)^2}{2}$

**প্রমাণ:** সূত্র ১ ও সূত্র ২ হতে,

$$a^2 + 2ab + b^2 = (a+b)^2$$
$$a^2 - 2ab + b^2 = (a-b)^2$$

যোগ করে, $2a^2 + 2b^2 = (a+b)^2 + (a-b)^2$

বা, $2(a^2 + b^2) = (a+b)^2 + (a-b)^2$

সুতরাং, $(a^2 + b^2) = \dfrac{(a+b)^2 + (a-b)^2}{2}$

**অনুসিদ্ধান্ত ৬.** $\quad ab = \left(\dfrac{a+b}{2}\right)^2 - \left(\dfrac{a-b}{2}\right)^2$

**প্রমাণ:** সূত্র ১ ও সূত্র ২ হতে,

$$a^2 + 2ab + b^2 = (a+b)^2$$
$$a^2 - 2ab + b^2 = (a-b)^2$$

বিয়োগ করে, $4ab = (a+b)^2 - (a-b)^2$

বা, $ab = \dfrac{(a+b)^2}{4} - \dfrac{(a-b)^2}{4}$

সুতরাং, $ab = \left(\dfrac{a+b}{2}\right)^2 - \left(\dfrac{a-b}{2}\right)^2$

**মন্তব্য:** অনুসিদ্ধান্ত ৬ প্রয়োগ করে যেকোনো দুইটি রাশির গুণফলকে ঐ দুইটি রাশির সমষ্টির অর্ধেকের ও দুইটির অন্তরের অর্ধেকের বর্গের অন্তররূপে প্রকাশ করা যায়।

**সূত্র ৩.** $\quad a^2 - b^2 = (a+b)(a-b)$

অর্থাৎ, দুইটি রাশির বর্গের বিয়োগফল = রাশি দুইটির যোগফল $\times$ রাশি দুইটির বিয়োগফল

**সূত্র ৪.** $\quad (x+a)(x+b) = x^2 + (a+b)x + ab$

অর্থাৎ, $(x+a)(x+b) = x^2 + (a$ ও $b$ এর বীজগাণিতিক যোগফল$)x + (a$ ও $b$ এর গুণফল$)$

**বর্ণসূত্রের সম্প্রসারণ:** $a + b + c$ রাশিটিতে তিনটি পদ আছে। এতে $(a+b)$ এবং $c$ এই দুইটি পদের সমষ্টিরূপে বিবেচনা করা যায়। অতএব, সূত্র ১ প্রয়োগে করে রাশিটির বর্গ করে পাই,

$(a + b + c)^2 = \{(a+b) + c\}^2 = (a+b)^2 + 2(a+b)c + c^2$
$= a^2 + 2ab + b^2 + 2ac + 2bc + c^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ac$

**সূত্র ৫.** $\quad (a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ac$

**অনুসিদ্ধান্ত ৭.** $\quad a^2 + b^2 + c^2 = (a + b + c)^2 - 2(ab + bc + ac)$

**অনুসিদ্ধান্ত ৮.** $\quad 2(ab + bc + ac) = (a + b + c)^2 - (a^2 + b^2 + c^2)$

**দ্রষ্টব্য:** সূত্র ৫ প্রয়োগ করে পাই,

ক) $(a + b - c)^2 = \{a + b + (-c)\}^2$
$= a^2 + b^2 + (-c)^2 + 2ab + 2b(-c) + 2a(-c)$
$= a^2 + b^2 + c^2 + 2ab - 2bc - 2ac$

খ) $(a - b + c)^2 = \{a + (-b) + c\}^2$
$= a^2 + (-b)^2 + c^2 + 2a(-b) + 2(-b)c + 2ac$
$= a^2 + b^2 + c^2 - 2ab - 2bc + 2ac$

গ) $(a - b - c)^2 = \{a + (-b) + (-c)\}^2$
$= a^2 + (-b)^2 + (-c)^2 + 2a(-b) + 2(-b)(-c) + 2a(-c)$
$= a^2 + b^2 + c^2 - 2ab + 2bc - 2ac$

### উদাহরণ ১.

$(4x + 5y)$ এর বর্গ কত?

**সমাধান:** $(4x + 5y)^2 = (4x)^2 + 2 \times (4x) \times (5y) + (5y)^2 = 16x^2 + 40xy + 25y^2$

### উদাহরণ ২.

$(3a - 7b)$ এর বর্গ কত?

**সমাধান:** $(3a - 7b)^2 = (3a)^2 - 2 \times (3a) \times (7b) + (7b)^2 = 9a^2 - 42ab + 49b^2$

### উদাহরণ ৩.

বর্গের সূত্র প্রয়োগ করে $996$ এর বর্গ নির্ণয় করো।

**সমাধান:** $(996)^2 = (1000 - 4)^2 = (1000)^2 - 2 \times 1000 \times 4 + 4^2$
$= 1000000 - 8000 + 16 = 1000016 - 8000 = 992016$

### উদাহরণ ৪.

$a + b + c + d$ এর বর্গ কত?

**সমাধান:** $(a + b + c + d)^2 = \{(a + b) + (c + d)\}^2$
$= (a + b)^2 + 2(a + b)(c + d) + (c + d)^2$
$= a^2 + 2ab + b^2 + 2(ac + ad + bc + bd) + c^2 + 2cd + d^2$
$= a^2 + b^2 + c^2 + d^2 + 2ab + 2ac + 2ad + 2bc + 2bd + 2cd$

> **কাজ:** সূত্রের সাহায্যে বর্গ নির্ণয় করো:
>- ক) $3xy + 2ax$
>- খ) $4x - 3y$
>- গ) $x - 5y + 2z$

### উদাহরণ ৫.

সরল করো:

$(5x + 7y + 3z)^2 + 2(7x - 7y - 3z)(5x + 7y + 3z) + (7x - 7y - 3z)^2$

**সমাধান:** ধরি, $5x + 7y + 3z = a$ এবং $7x - 7y - 3z = b$

$\therefore$ প্রদত্ত রাশি $= a^2 + 2 \cdot b \cdot a + b^2 = a^2 + 2ab + b^2$
$\qquad = (a + b)^2$
$\qquad = \{(5x + 7y + 3z) + (7x - 7y - 3z)\}^2 \qquad [\,a$ ও $b$ এর মান বসিয়ে$\,]$
$\qquad = (5x + 7y + 3z + 7x - 7y - 3z)^2$
$\qquad = (12x)^2 = 144x^2$

### উদাহরণ ৬.

$x - y = 2$ এবং $xy = 24$ হলে, $x + y$ এর মান কত?

**সমাধান:** $(x + y)^2 = (x - y)^2 + 4xy = (2)^2 + 4 \times 24 = 4 + 96 = 100$

$\therefore x + y = \pm \sqrt{100} = \pm 10$

### উদাহরণ ৭.

যদি $a^4 + a^2 b^2 + b^4 = 3$ এবং $a^2 + b^2 = 3$ হয়, তবে $a^2 + b^2$ এর মান কত?

**সমাধান:** $a^4 + a^2 b^2 + b^4$
$= (a^2)^2 + 2a^2 b^2 + (b^2)^2 - a^2 b^2$
$= (a^2 + b^2)^2 - (ab)^2$
$= (a^2 + b^2 + ab)(a^2 + b^2 - ab)$
$= (a^2 + ab + b^2)(a^2 - ab + b^2)$

$\therefore 3 = 3(a^2 - ab + b^2) \quad [$ মান বসিয়ে $]$

বা, $a^2 - ab + b^2 = \dfrac{3}{3} = 1$

এখন, $a^2 - ab + b^2 = 3$ এবং $a^2 - ab + b^2 = 1$

যোগ করে পাই, $2(a^2 + b^2) = 4$

বা, $a^2 + b^2 = \dfrac{4}{2} = 2$

$\therefore a^2 + b^2 = 2$

### উদাহরণ ৮.

প্রমাণ করো যে, $(a + b)^4 - (a - b)^4 = 8ab(a^2 + b^2)$

**সমাধান:** $(a + b)^4 - (a - b)^4$
$= \{(a + b)^2\}^2 - \{(a - b)^2\}^2$
$= \{(a + b)^2 + (a - b)^2\}\{(a + b)^2 - (a - b)^2\}$
$= 2(a^2 + b^2) \times 4ab \quad [$ অনুসিদ্ধান্ত ৫ এবং অনুসিদ্ধান্ত ৬ ব্যবহার করে $]$
$= 8ab(a^2 + b^2)$

$\therefore (a + b)^4 - (a - b)^4 = 8ab(a^2 + b^2)$

### উদাহরণ ৯.

$a + b + c = 15$ এবং $a^2 + b^2 + c^2 = 83$ হলে, $ab + bc + ac$ এর মান কত?

**সমাধান:** প্রথম পদ্ধতি:

$2(ab + bc + ac) = (a + b + c)^2 - (a^2 + b^2 + c^2) = (15)^2 - 83 = 225 - 83 = 142$

$\therefore ab + bc + ac = \dfrac{142}{2} = 71$

বিকল্প পদ্ধতি:

$(a + b + c)^2 = (a^2 + b^2 + c^2) + 2(ab + bc + ac)$

বা, $(15)^2 = 83 + 2(ab + bc + ac)$

বা, $225 - 83 = 2(ab + bc + ac)$

বা, $2(ab + bc + ac) = 142$

$\therefore ab + bc + ac = \dfrac{142}{2} = 71$

### উদাহরণ ১০.

$a + b + c = 0$ এবং $ab + bc + ac = 1$ হলে, $(a + b)^2 + (b + c)^2 + (c + a)^2$ এর মান কত?

**সমাধান:** $(a + b)^2 + (b + c)^2 + (c + a)^2$
$= a^2 + 2ab + b^2 + b^2 + 2bc + c^2 + c^2 + 2ca + a^2$
$= (a^2 + b^2 + c^2 + 2ab + 2bc + 2ca) + (a^2 + b^2 + c^2)$
$= (a + b + c)^2 + (a + b + c)^2 - 2(ab + bc + ca)$
$= (2)^2 + (2)^2 - 2 \times 1 = 4 + 4 - 2 = 8 - 2 = 6$

### উদাহরণ ১১.

**সমাধান:** ধরি, $2x + 3y = a$ এবং $4x - 5y = b$

$\therefore$ প্রদত্ত রাশি $ab = \left(\dfrac{a+b}{2}\right)^2 - \left(\dfrac{a-b}{2}\right)^2$
$\qquad = \left(\dfrac{2x + 3y + 4x - 5y}{2}\right)^2 - \left(\dfrac{2x + 3y - 4x + 5y}{2}\right)^2 \quad [a$ ও $b$ এর মান বসিয়ে$]$

$\qquad = (3x - y)^2 - (4y - x)^2$

## ৩.৩ ঘন সংবলিত সূত্রাবলি

**সূত্র ৬.** $\quad (a + b)^3 = a^3 + 3a^2 b + 3a b^2 + b^3 = a^3 + b^3 + 3ab(a + b)$

**প্রমাণ:** $(a + b)^3 = (a + b)(a + b)^2$
$= (a + b)(a^2 + 2ab + b^2)$
$= a(a^2 + 2ab + b^2) + b(a^2 + 2ab + b^2)$
$= a^3 + 2a^2 b + ab^2 + a^2 b + 2ab^2 + b^3$
$= a^3 + 3a^2 b + 3ab^2 + b^3$
$= a^3 + b^3 + 3ab(a + b)$

**অনুসিদ্ধান্ত ৯.** $\quad a^3 + b^3 = (a + b)^3 - 3ab(a + b)$

**সূত্র ৭.** $\quad (a - b)^3 = a^3 - 3a^2 b + 3ab^2 - b^3 = a^3 - b^3 - 3ab(a - b)$

**প্রমাণ:** $(a - b)^3 = (a - b)(a - b)^2$
$= (a - b)(a^2 - 2ab + b^2)$
$= a(a^2 - 2ab + b^2) - b(a^2 - 2ab + b^2)$
$= a^3 - 2a^2 b + ab^2 - a^2 b + 2ab^2 - b^3$
$= a^3 - 3a^2 b + 3ab^2 - b^3$
$= a^3 - b^3 - 3ab(a - b)$

**দ্রষ্টব্য:** সূত্র ৬ এ $b$ এর স্থলে $-b$ বসালে সূত্র ৭ পাওয়া যায়:

$\{a + (-b)\}^3 = a^3 + (-b)^3 + 3a\{a + (-b)\}$

অর্থাৎ, $(a - b)^3 = a^3 - b^3 - 3ab(a - b)$

**অনুসিদ্ধান্ত ১০.** $\quad a^3 - b^3 = (a - b)^3 + 3ab(a - b)$

**সূত্র ৮.** $\quad a^3 + b^3 = (a + b)(a^2 - ab + b^2)$

**প্রমাণ:** $a^3 + b^3 = (a + b)^3 - 3ab(a + b)$
$= (a + b)\{(a + b)^2 - 3ab\}$
$= (a + b)(a^2 + 2ab + b^2 - 3ab)$
$= (a + b)(a^2 - ab + b^2)$

**সূত্র ৯.** $\quad a^3 - b^3 = (a - b)(a^2 + ab + b^2)$

**প্রমাণ:** $a^3 - b^3 = (a - b)^3 + 3ab(a - b)$
$= (a - b)\{(a - b)^2 + 3ab\}$
$= (a - b)(a^2 - 2ab + b^2 + 3ab)$
$= (a - b)(a^2 + ab + b^2)$

## ৩.৪ উৎপাদকে বিশ্লেষণ (Factorization)

কোনো রাশি দুই বা ততোধিক রাশির গুণফলের সমান হলে, শেষোক্ত রাশিগুলোর প্রত্যেকটিকে প্রথমোক্ত রাশির **উৎপাদক** বা **গুণনীয়ক** বলা হয়। কোনো বীজগাণিতিক রাশির উৎপাদকগুলো নির্ণয় করা বা রাশিটিকে দুই বা ততোধিক রাশির গুণফলরূপে প্রকাশ করাকে রাশিটির **উৎপাদকে বিশ্লেষণ** বলা হয়। বীজগাণিতিক রাশিগুলো একই প্রকার পদবিশিষ্ট (বহুপদী) হতে পারে। সেরূপ উক্ত রাশির উৎপাদকগুলোও এক বা একাধিক পদবিশিষ্টও হতে পারে। এখানে উৎপাদক নির্ণয়ের কতিপয় কৌশল আলোচনা করা হলো।

**সাধারণ পদ্ধতি:** কোনো বহুপদীর প্রত্যেক পদে কোনো সাধারণ উৎপাদক থাকলে তা বের করে নিতে হয়। যেমন:

### উদাহরণ ১৮.

$3a^2 b + 6a b^2 + 12a^3 b^2 = 3ab(a + 2b + 4ab)$

### উদাহরণ ১৯.

$2ab(x - y) + 2bc(x - y) + 3ca(x - y) = (x - y)(2ab + 2bc + 3ca)$

**পূর্ণবর্গ:** একটি রাশিকে পূর্ণবর্গ আকারে প্রকাশ করেও উৎপাদকে বিশ্লেষণ করা যায়।

### উদাহরণ ২০.

$4x^2 + 12x + 9$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $4x^2 + 12x + 9 = (2x)^2 + 2 \times 2x \times 3 + (3)^2$
$= (2x + 3)^2 = (2x + 3)(2x + 3)$

### উদাহরণ ২১.

$9x^2 - 30xy + 25y^2$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $9x^2 - 30xy + 25y^2$
$= (3x)^2 - 2 \times 3x \times 5y + (5y)^2$
$= (3x - 5y)^2 = (3x - 5y)(3x - 5y)$

**দুইটি বর্গের অন্তর:** একটি রাশিকে দুইটি বর্গের অন্তররূপে প্রকাশ করে এবং $a^2 - b^2 = (a + b)(a - b)$ সূত্র প্রয়োগ করেও উৎপাদকে বিশ্লেষণ করা যায়।

### উদাহরণ ২২.

$a^2 - 1 + 2b - b^2$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $a^2 - 1 + 2b - b^2 = a^2 - (b^2 - 2b + 1)$
$= a^2 - (b - 1)^2 = \{a + (b - 1)\}\{a - (b - 1)\}$
$= (a + b - 1)(a - b + 1)$

### উদাহরণ ২৩.

$a^4 + 64b^4$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $a^4 + 64b^4 = (a^2)^2 + (8b^2)^2$
$= (a^2)^2 + 2 \times a^2 \times 8b^2 + (8b^2)^2 - 16a^2 b^2$
$= (a^2 + 8b^2)^2 - (4ab)^2$
$= (a^2 + 8b^2 + 4ab)(a^2 + 8b^2 - 4ab)$
$= (a^2 + 4ab + 8b^2)(a^2 - 4ab + 8b^2)$

> **কাজ:** উৎপাদকে বিশ্লেষণ করো:
>- ক) $abx^2 + acx^3 + adx^4$
>- খ) $xa^2 - 144xb^2$
>- গ) $x^2 - 2xy - 4y - 4$

**সরল মধ্যপদ বিভক্তিকরণ:** $x^2 + (a + b)x + ab = (x + a)(x + b)$ সূত্রটি ব্যবহার করে উৎপাদকে নির্ণয় করা যায়। এ পদ্ধতিতে $x^2 + px + q$ আকারের বহুপদীর উৎপাদক নির্ণয় করা সম্ভব হবে যদি দুইটি সংখ্যা $a$ ও $b$ নির্ণয় করা যায়, যেন $a + b = p$ এবং $ab = q$ হয়। এজন্য $q$ এর উৎপাদক নিতে হয় যাদের বীজগাণিতিক সমষ্টি $p$ হয়। $q > 0$ হলে, $a$ ও $b$ একই চিহ্নযুক্ত হবে এবং $q < 0$ হলে, $a$ ও $b$ বিপরীত চিহ্নযুক্ত হবে। উল্লেখ্য $p$ এবং $q$ পূর্ণসংখ্যা না-ও হতে পারে।

### উদাহরণ ২৪.

$x^2 + 12x + 35$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $x^2 + 12x + 35 = x^2 + (5 + 7)x + 5 \times 7 = (x + 5)(x + 7)$

### উদাহরণ ২৫.

$x^2 + x - 20$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $x^2 + x - 20 = x^2 + (5 - 4)x + (5)(-4) = (x + 5)(x - 4)$

**যৌগিক মধ্যপদ বিভক্তিকরণ:** $ax^2 + bx + c$ আকারের বহুপদীর মধ্যপদ বিভক্তিকরণ পদ্ধতিতে $ax^2 + bx + c = (rx + p)(sx + q)$ হবে যদি $ax^2 + bx + c = rsx^2 + (rq + sp)x + pq$ হয়। অর্থাৎ, $a = rs, b = rq + sp$ এবং $c = pq$। সুতরাং, $ac = rspq = (rp)(sq)$ অতএব, $ax^2 + bx + c$ আকারের বহুপদীর উৎপাদকে বিশ্লেষণ করতে হবে $ac$, যদি সহগ এবং $x$ এর সহগকে এমন দুইটি উৎপাদকে প্রকাশ করতে হবে, যাদের বীজগাণিতিক সমষ্টি $x$ এর সহগ $b$ এর সমান হয়।

### উদাহরণ ২৬.

$3x^2 - x - 14$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $3x^2 - x - 14 = 3x^2 - 7x + 6x - 14$
$\qquad = x(3x - 7) + 2(3x - 7) = (3x - 7)(x + 2)$

> **কাজ:** উৎপাদকে বিশ্লেষণ করো:
>- ক) $x^2 + x - 56$
>- খ) $16x^3 - 46x^2 + 15x$
>- গ) $12x^2 + 17x + 6$

**ঘন আকার:** একটি রাশিকে পূর্ণঘন আকারে প্রকাশ করেও উৎপাদক নির্ণয় করা যায়।

### উদাহরণ ২৭.

$8x^3 + 36x^2 y + 54xy^2 + 27y^3$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $8x^3 + 36x^2 y + 54xy^2 + 27y^3$ এর বিয়োগফলের সূত্র দিয়ে:
$= (2x)^3 + 3 \times (2x)^2 \times 3y + 3 \times 2x \times (3y)^2 + (3y)^3$
$= (2x + 3y)^3 = (2x + 3y)(2x + 3y)(2x + 3y)$

**দুইটি ঘন এর যোগফল বা বিয়োগফলের সূত্র দিয়ে:** $a^3 + b^3 = (a + b)(a^2 - ab + b^2)$ এবং $a^3 - b^3 = (a - b)(a^2 + ab + b^2)$ সূত্র দুইটি ব্যবহার করে উৎপাদক নির্ণয় করা যায়।

### উদাহরণ ২৮.

উৎপাদকে বিশ্লেষণ করো: ক) $8a^3 + 27b^3$ খ) $a^6 - 64$

**সমাধান:**

ক) $8a^3 + 27b^3 = (2a)^3 + (3b)^3$
$= (2a + 3b)\{(2a)^2 - 2a \times 3b + (3b)^2\}$
$= (2a + 3b)(4a^2 - 6ab + 9b^2)$

খ) $a^6 - 64 = (a^2)^3 - (4)^3 = (a^2 - 4)\{(a^2)^2 + a^2 \times 4 + (4)^2\}$
$= (a^2 - 4)(a^4 + 4a^2 + 16)$
কিন্তু $a^2 - 4 = a^2 - 2^2 = (a + 2)(a - 2)$
এবং $a^4 + 4a^2 + 16 = (a^2)^2 + (4)^2 + 4a^2$
$= (a^2 + 4)^2 - 2(a^2)(4) + 4a^2$
$= (a^2 + 4)^2 - 4a^2$
$= (a^2 + 4)^2 - (2a)^2$
$= (a^2 + 4 + 2a)(a^2 + 4 - 2a)$
$= (a^2 + 2a + 4)(a^2 - 2a + 4)$
$\therefore a^6 - 64 = (a + 2)(a - 2)(a^2 + 2a + 4)(a^2 - 2a + 4)$

বিকল্প নিয়ম: $a^6 - 64 = (a^3)^2 - 8^2$
$= (a^3 + 8)(a^3 - 8)$
$= (a^3 + 2^3)(a^3 - 2^3)$
$= (a + 2)(a^2 - 2a + 4)(a - 2)(a^2 + 2a + 4)$

> **কাজ:** উৎপাদকে বিশ্লেষণ করো:
>- ক) $2x^4 + 16x$
>- খ) $8 - a^3 + 3a^2 b - 3ab^2 + b^3$
>- গ) $(a + b)^3 + (a - b)^3$

**ভগ্নাংশসহযুক্ত রাশির উৎপাদক:** ভগ্নাংশসহযুক্ত রাশির উৎপাদকগুলোকে বিভিন্নভাবে প্রকাশ করা যায়। যেমন, $a^3 + \dfrac{1}{27} = a^3 + \dfrac{1}{3^3} = \left(a + \dfrac{1}{3}\right)\left(a^2 - \dfrac{a}{3} + \dfrac{1}{9}\right)$

আবার, $a^3 + \dfrac{1}{27} = \dfrac{1}{27}(27a^3 + 1) = \dfrac{1}{27}\{(3a)^3 + (1)^3\} = \dfrac{1}{27}(3a + 1)(9a^2 - 3a + 1)$

দ্বিতীয় সমাধানে চলক-সংশ্লিষ্ট উৎপাদকগুলো সহগসুদ্ধ পূর্ণসংখ্যা কিন্তু সমাধান দুইটিই অভিন্ন।

$\dfrac{1}{27}(3a + 1)(9a^2 - 3a + 1) = \dfrac{1}{3}(3a + 1) \times \dfrac{1}{9}(9a^2 - 3a + 1)$
$= \left(a + \dfrac{1}{3}\right)\left(a^2 - \dfrac{a}{3} + \dfrac{1}{9}\right)$

### উদাহরণ ২৯.

$x^3 + 6x^2 y + 11xy^2 + 6y^3$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** $x^3 + 6x^2 y + 11xy^2 + 6y^3$
$= \{x^3 + 3 \cdot x^2 \cdot 2y + 3 \cdot x \cdot (2y)^2 + (2y)^3\} - xy^2 - 2y^3$
$= (x + 2y)^3 - y^2(x + 2y)$
$= (x + 2y)\{(x + 2y)^2 - y^2\}$
$= (x + 2y)(x + 2y + y)(x + 2y - y)$
$= (x + 2y)(x + 3y)(x + y) = (x + y)(x + 2y)(x + 3y)$

> **কাজ:** উৎপাদকে বিশ্লেষণ করো:
>- ক) $\dfrac{1}{2}x^2 + \dfrac{7}{6}x + \dfrac{1}{3}$
>- খ) $a^3 + \dfrac{1}{8}$
>- গ) $16x^2 - 25y^2 - 8xz + 10yz$

## ৩.৫ ভাগশেষ উপপাদ্য (Remainder Theorem)

নিচের উদাহরণটিতে $6x^2 - 7x + 5$ কে $x - 1$ দ্বারা ভাগ করলে ভাগফল ও ভাগশেষ কত?

$$\begin{array}{r|l}
x - 1 & 6x^2 - 7x + 5 \quad (6x - 1 \\
& \underline{6x^2 - 6x} \\
& \phantom{6x^2}\;\, -x + 5 \\
& \phantom{6x^2}\;\, \underline{-x + 1} \\
& \phantom{6x^2 - 0x +}\, 4
\end{array}$$

এখানে, ভাজক $x - 1$, ভাজ্য $6x^2 - 7x + 5$, ভাগফল $6x - 1$ এবং ভাগশেষ $4$।

আমরা জানি, ভাজ্য = ভাজক $\times$ ভাগফল + ভাগশেষ।

এখন যদি আমরা ভাজ্যকে $f(x)$, ভাগফলকে $h(x)$, ভাগশেষকে $r$ ও ভাজককে $(x - a)$ দ্বারা সূচিত করি, তাহলে উপরের সূত্র মতে পাই,

$f(x) = (x - a) \cdot h(x) + r$, এই সূত্রটি $a$ এর সকল মানের জন্য সত্য।

উভয়পক্ষে $x = a$ বসিয়ে পাই,

$f(a) = (a - a) \cdot h(a) + r = 0 \cdot h(a) + r = r$

সুতরাং, $r = f(a)$

অতএব, $f(x)$ কে $(x - a)$ দ্বারা ভাগ করলে ভাগশেষ হয় $f(a)$। এই সূত্র ভাগশেষ উপপাদ্য (Remainder theorem) নামে পরিচিত। অর্থাৎ, ধনাত্মক মাত্রার কোনো বহুপদী $f(x)$ কে $(x - a)$ আকারের বহুপদী দ্বারা ভাগ করলে ভাগশেষ কত হবে তা ভাগ না করে বের করার সূত্রই হলো ভাগশেষ উপপাদ্য। উপরের উদাহরণে $a = 1$ হলে $f(x) = 6x^2 - 7x + 5$।

$\therefore f(1) = 6 - 7 + 5 = 4$ অতএব, $f(x)$ ভাগশেষের সমান। ভাজক $(x - 1)$ এর মাত্রা $1$, ভাজক রাজনে উৎপাদক হয়, তাহলে ভাগশেষ হবে শূন্য। আর যদি উৎপাদক না হয়, তাহলে ভাগশেষ থাকবে এবং তা হবে অশূন্য কোনো সংখ্যা। তবে সাধারণভাবে বলতে গেলে ভাগফল ভাজকের থেকে কম মাত্রার একটি বহুপদী হবে।

**অনুসিদ্ধান্ত ১১.** $(x - a), f(x)$ এর উৎপাদক হবে, যদি এবং কেবল যদি $f(a) = 0$ হয়।

**প্রমাণ:** ধরি, $f(a) = 0$ অতএব, ভাগশেষ উপপাদ্য অনুযায়ী, $f(x)$ কে $(x - a)$ দ্বারা ভাগ করলে ভাগশেষ শূন্য হবে। অর্থাৎ, $(x - a), f(x)$ এর একটি উৎপাদক হবে।

বিপরীতক্রমে, ধরি, $(x - a), f(x)$ এর একটি উৎপাদক।

অতএব, $f(x) = (x - a) \cdot h(x)$, যেখানে $h(x)$ বহুপদী।

উভয়পক্ষে $x = a$ বসিয়ে পাই,

$f(a) = (a - a) \cdot h(a) = 0$

$\therefore f(a) = 0$

সুতরাং, কোনো বহুপদী $f(x), (x - a)$ দ্বারা বিভাজ্য হবে যদি এবং কেবল যদি $f(a) = 0$ হয়। এই সূত্র উৎপাদক উপপাদ্য (Factor theorem) নামে পরিচিত।

**প্রতিজ্ঞা ১২.** যদি $f(x)$ এর মাত্রা ধনাত্মক হয় এবং $a \neq 0$ হয়, তবে $f(x)$ কে $(ax + b)$ দ্বারা ভাগ করলে ভাগশেষ হয় $f\left(-\dfrac{b}{a}\right)$।

**প্রমাণ:** ভাজক $ax + b, (a \neq 0)$ এর মাত্রা $1$।

সুতরাং আমরা লিখতে পারি, $f(x) = (ax + b) \cdot h(x) + r = a\left(x + \dfrac{b}{a}\right) \cdot h(x) + r$

$\therefore f(x) = \left(x + \dfrac{b}{a}\right) \cdot a \cdot h(x) + r$

দেখা যাচ্ছে যে, $f(x)$ কে $\left(x + \dfrac{b}{a}\right)$ দ্বারা ভাগ করলে ভাগফল হয়, $a \cdot h(x)$ এবং ভাগশেষ হয় $r$।

এখানে, ভাজক $= x - \left(-\dfrac{b}{a}\right)$

সুতরাং ভাগশেষ উপপাদ্য অনুযায়ী, $r = f\left(-\dfrac{b}{a}\right)$

অতএব, $f(x)$ কে $(ax + b)$ দ্বারা ভাগ করলে ভাগশেষ হয় $f\left(-\dfrac{b}{a}\right)$।

**অনুসিদ্ধান্ত ১৩.** $ax + b, a \neq 0$ হলে, রাশিটি কোনো বহুপদী $f(x)$ এর উৎপাদক হবে, যদি এবং কেবল যদি $f\left(-\dfrac{b}{a}\right) = 0$ হয়।

**প্রমাণ:** $a \neq 0, ax + b = a\left(x + \dfrac{b}{a}\right), f(x)$ এর উৎপাদক হবে, যদি এবং কেবল যদি $\left(x + \dfrac{b}{a}\right) = x - \left(-\dfrac{b}{a}\right), f(x)$ এর একটি উৎপাদক হবে। অর্থাৎ, যদি এবং কেবল যদি $f\left(-\dfrac{b}{a}\right) = 0$ হয়।

ভাগশেষ উপপাদ্যের সাহায্যে উৎপাদক নির্ণয়ের এই পদ্ধতিকে **শূন্যায়ন পদ্ধতি (Vanishing method)** বলে।

### উদাহরণ ৩০.

$x^3 - x - 6$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** এখানে, $f(x) = x^3 - x - 6$ একটি বহুপদী। এর ধ্রুবপদ $-6$ এর উৎপাদকগুলো হচ্ছে $\pm 1, \pm 2, \pm 3, \pm 6$।

এখন, $x = 1, -1$ বসিয়ে দেখি, $f(x)$ এর মান শূন্য হয় না।

কিন্তু $x = 2$ বসিয়ে দেখি, $f(x)$ এর মান শূন্য হয়।

অর্থাৎ, $f(2) = 2^3 - 2 - 6 = 8 - 2 - 6 = 0$।

সুতরাং, $x - 2, f(x)$ বহুপদীটির একটি উৎপাদক।

$\therefore f(x) = x^3 - x - 6$
$\qquad = x^3 - 2x^2 + 2x^2 - 4x + 3x - 6$
$\qquad = x^2(x - 2) + 2x(x - 2) + 3(x - 2)$
$\qquad = (x - 2)(x^2 + 2x + 3)$

### উদাহরণ ৩১.

$x^3 - 3xy^2 + 2y^3$ এবং $x^2 + xy - 2y^2$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** এখানে, $x$ কে চলক এবং $y$ কে ধ্রুবক হিসেবে বিবেচনা করি।

প্রদত্ত রাশিকে $x$-এর বহুপদী বিবেচনা করে

ধরি, $f(x) = x^3 - 3xy^2 + 2y^3$

তাহলে, $f(y) = y^3 - 3y \cdot y^2 + 2y^3 = 3y^3 - 3y^3 = 0$

$\therefore (x - y), f(x)$ এর একটি উৎপাদক।

এখন, $x^3 - 3xy^2 + 2y^3$
$= x^3 - x^2 y + x^2 y - xy^2 - 2xy^2 + 2y^3$
$= x^2(x - y) + xy(x - y) - 2y^2(x - y) = (x - y)(x^2 + xy - 2y^2)$

আবার ধরি, $g(y) = x^2 + xy - 2y^2$

$\therefore g(y) = y^2 + y^2 - 2y^2 = 0$

$\therefore (x - y), g(x)$ এর একটি উৎপাদক

$\therefore g(x) = x^2 + xy - 2y^2$
$\qquad = x^2 - xy + 2xy - 2y^2$
$\qquad = x(x - y) + 2y(x - y)$
$\qquad = (x - y)(x + 2y)$
$\therefore x^3 - 3xy^2 + 2y^3 = (x - y)^2 (x + 2y)$

### উদাহরণ ৩২.

$54x^4 + 27x^3 a - 16x - 8a$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** ধরি, $f(x) = 54x^4 + 27x^3 a - 16x - 8a$

তাহলে, $f\left(-\dfrac{1}{2}a\right) = 54\left(-\dfrac{1}{2}a\right)^4 + 27a\left(-\dfrac{1}{2}a\right)^3 - 16\left(-\dfrac{1}{2}a\right) - 8a$
$\qquad = \dfrac{27}{8}a^4 - \dfrac{27}{8}a^4 + 8a - 8a = 0$

$\therefore x - \left(-\dfrac{1}{2}a\right) = x + \dfrac{a}{2} = \dfrac{1}{2}(2x + a), f(x)$ এর একটি উৎপাদক

অর্থাৎ, $(2x + a), f(x)$ এর একটি উৎপাদক।

এখন, $54x^4 + 27x^3 a - 16x - 8a$
$= 27x^3(2x + a) - 8(2x + a)$
$= (2x + a)(27x^3 - 8)$
$= (2x + a)\{(3x)^3 - (2)^3\}$
$= (2x + a)(3x - 2)(9x^2 + 6x + 4)$

### উদাহরণ ৩৩.

$g(a) = a^3 + a^2 + 10a - 8, f(a) = a^3 - 9 + (a + 1)^3$।
- ক) $g(a)$ কে $(a - 2)$ দ্বারা ভাগ করলে ভাগশেষ কত হবে তা নির্ণয় করো।
- খ) $f(a)$ কে উৎপাদকে বিশ্লেষণ করো।

**সমাধান:** ক) দেওয়া আছে, $g(a) = a^3 + a^2 + 10a - 8$

ভাগশেষ উপপাদ্য অনুসারে $g(a)$ কে $(a - 2)$ দ্বারা ভাগ করলে ভাগশেষ হবে $g(2)$।

$\therefore g(2) = 2^3 + 2^2 + 10 \cdot 2 - 8 = 8 + 4 + 20 - 8 = 32 - 8 = 24$

$\therefore$ নির্ণেয় ভাগশেষ $24$

খ) $f(a) = a^3 - 9 + (a + 1)^3$ বহুপদীর মান শূন্য হয়।

$f(a)$ একটি বহুপদী, $a = 1$ বসালে বহুপদীটির মান শূন্য হয়।

ফলে $(a - 1)$ একটি উৎপাদক।

$\therefore f(a) = a^3 - 9 + 3a^2 + 3a + 1 = 2a^3 + 3a^2 + 3a - 8$

$= 2a^3 - 2a^2 + 5a^2 - 5a + 8a - 8$
$= 2a^2(a - 1) + 5a(a - 1) + 8(a - 1)$
$= (a - 1)(2a^2 + 5a + 8)$

$\therefore a^3 - 9 + (a + 1)^3 = (a - 1)(2a^2 + 5a + 8)$

>- ক) $x^3 - 21x - 20$
>- খ) $2x^3 - 3x^2 + 3x - 1$
>- গ) $x^3 + 6x^2 + 11x + 6$

## ৩.৬ বাস্তব সমস্যা সমাধানে বীজগাণিতিক সূত্র গঠন ও প্রয়োগ

দৈনন্দিন কাজে বিভিন্ন সময়ে আমরা বাস্তব সমস্যার সম্মুখীন হই। এই সমস্যাগুলো ভাগ্যবচনে বর্ণিত হয়। এ অনুচ্ছেদে আমরা ভাগ্যবচনে বর্ণিত বাস্তব পরিবেশের বিভিন্ন সমস্যা সমাধানপক্ষে বীজগাণিতিক সূত্র গঠন এবং সূত্র প্রয়োগ করার পদ্ধতি নিয়ে আলোচনা করব। সমাজে কীভাবে কোন বীজগাণিতিক সূত্র গঠন করতে হবে এদের ব্যবহার করতে হবে তা নিয়ে আলোচনা পরিচালিত হবে। শিক্ষার্থীরা এতে চলিত শ্রেণিতে গণিতের প্রয়োগ সম্পর্কে অর্থপূর্ণ চিত্র পাবে, অন্যদিকে দৈনন্দিন পারিপার্শ্বিক অবস্থায় গণিতের সম্পৃক্ততা বুঝতে গেলে গণিত শিক্ষার প্রতি আগ্রহী হবে।

**সমস্যা সমাধানের পদ্ধতি:**

১. প্রথমেই সতর্কতার সাথে সমস্যাটি পর্যবেক্ষণ করে এবং মনোযোগ সহকারে পড়ে কোনগুলো অজ্ঞাত এবং কী দ্বারা নির্ণয় করতে হবে তা চিহ্নিত করতে হবে।

২. অজ্ঞাত সংখ্যা বা সংখ্যা সমূহের যেকোনো একটিকে চলক (ধরি $x$) দ্বারা সূচিত করতে হবে। অতঃপর সমস্যাটি ভাষ্যানুযায়ী অনুধাবন করে সম্ভব হলে অন্যান্য অজ্ঞাত রাশিগুলোকেও একই চলক $x$ এর মাধ্যমে প্রকাশ করতে হবে।

৩. সমস্যাকে ক্ষুদ্র ক্ষুদ্র অংশে বিভক্ত করে বীজগাণিতিক রাশি দ্বারা প্রকাশ করতে হবে।

৪. প্রদত্ত শর্ত ব্যবহার করে ক্ষুদ্র ক্ষুদ্র অংশগুলোকে একত্রে সমীকরণ আকারে প্রকাশ করতে হবে।

৫. সমীকরণটি সমাধান করে অজ্ঞাত সংখ্যা $x$ এর মান নির্ণয় করতে হবে।

৬. প্রাপ্ত ফলাফলগুলো ভাষ্যানুসারে বর্ণিত মূল সমস্যার সর্বসদিক সিদ্ধ হয় কিনা তা পরীক্ষা করতে হবে।

বাস্তব সমস্যা সমাধানে বিভিন্ন সূত্র ব্যবহার করা হয়। সুবিধার্থে এখানে আলোচনা করা হলো।

**দেয় বা প্রাপ্য বিষয়ক**

মনে করি, $q$ = জনপ্রতি দেয় বা প্রাপ্য টাকার পরিমাণ
$\qquad\quad n$ = লোকের সংখ্যা

$\therefore$ দেয় বা প্রাপ্য টাকার পরিমাণ, $A = qn$

**সময় ও কাজ বিষয়ক**

মনে করি, $q$ = প্রত্যেকে একক সময়ে কাজের যে অংশ সম্পন্ন করে
$\qquad\quad n$ = কাজ সম্পাদনকারীর সংখ্যা
$\qquad\quad x$ = কাজের মোট সময়
$\qquad\quad W = n$ জনে $x$ সময়ে কাজের যে অংশ সম্পন্ন করে

$\therefore W = qnx$

**সময় ও দূরত্ব বিষয়ক**

মনে করি, $v$ = প্রতি ঘণ্টায় গতিবেগ
$\qquad\quad t$ = মোট সময়
$\qquad\quad d$ = মোট দূরত্ব

$\therefore d = vt$

**নল ও চৌবাচ্চা বিষয়ক**

মনে করি, $Q_0$ = নলের মুখ খুলে দেওয়ার সময় চৌবাচ্চায় জমা পানির পরিমাণ
$\qquad\quad q$ = প্রতি একক সময়ে নল দিয়ে যে পানি প্রবেশ করে অথবা বের হয়
$\qquad\quad t$ = অতিক্রান্ত সময়
$\qquad\quad Q(t) = t$ সময়ে চৌবাচ্চায় পানির পরিমাণ

$\therefore Q(t) = Q_0 \pm qt$

পানি প্রবেশ হওয়ার শর্তে '+' চিহ্ন এবং পানি বের হওয়ার শর্তে '−' চিহ্ন ব্যবহার করতে হবে।

**শতকরা অংশ বিষয়ক**

মনে করি, $b$ = মোট রাশি

$r$ = শতকরা হার $= \dfrac{s}{100} = s\%$

$p$ = শতকরা অংশ $= b$ এর $s\%$

$\therefore p = br$

**লাভ-ক্ষতি বিষয়ক**

মনে করি, $C$ = ক্রয়মূল্য

$r$ = লাভ বা ক্ষতির শতকরা হার

$\therefore$ বিক্রয়মূল্য $S = C(1 \pm r)$

লাভের ক্ষেত্রে, $S = C(1 + r)$ এবং ক্ষতির ক্ষেত্রে, $S = C(1 - r)$

**বিনিয়োগ-মুনাফা বিষয়ক**

মনে করি, $I = n$ একক সময় পরে মুনাফা
$\qquad\quad n$ = নির্দিষ্ট সংখ্যক একক সময়
$\qquad\quad P$ = মূলধনের পরিমাণ
$\qquad\quad r$ = একক সময়ে একক মূলধনের মুনাফা
$\qquad\quad A = n$ একক সময় পরে মুনাফাসহ মূলধন

সরল মুনাফার ক্ষেত্রে,

$\quad I = Pnr$

$\quad A = P + I = P + Pnr = P(1 + nr)$

চক্রবৃদ্ধি মুনাফার ক্ষেত্রে, $C = P(1 + r)^n$

### উদাহরণ ৩৪.

বার্ষিক ক্রীড়া অনুষ্ঠান করার জন্য কোনো এক সমিতির সদস্যরা $45,000$ টাকার বাজেট করলেন এবং সিদ্ধান্ত নিলেন যে, প্রত্যেক সদস্যই সমান চাঁদা দিবেন। কিন্তু $5$ জন সদস্য চাঁদা দিতে অসম্মতি জানালেন। এর ফলে প্রত্যেক সদস্যের মাথাপিছু $15$ টাকা চাঁদা বৃদ্ধি পেল। ঐ সমিতিতে কতজন সদস্য ভাই ছিলেন?

**সমাধান:** মনে করি, সমিতির সদস্য সংখ্যা $x$ এবং জনপ্রতি দেয় চাঁদার পরিমাণ $q$ টাকা। তাহলে, মোট চাঁদা, $A = qx = 45,000$ টাকা।

প্রকৃতপক্ষে চাঁদা প্রদানকারী সদস্য সংখ্যা ছিল $(x - 5)$ জন এবং জনপ্রতি চাঁদা $(q + 15)$ টাকা।

তাহলে, মোট চাঁদা হলো $(x - 5)(q + 15)$

প্রশ্নানুসারে,

$\quad qx = (x - 5)(q + 15) \cdots \cdots (1)$
$\quad qx = 45000 \cdots \cdots (2)$

সমীকরণ $(1)$ থেকে পাই,

$\quad qx = (x - 5)(q + 15)$

বা, $qx = qx + 15x - 5q - 75$

বা, $5q = 15x - 75 = 5(3x - 15)$

$\therefore q = 3x - 15$

সমীকরণ $(2)$ এ $q$ এর মান বসিয়ে পাই,

$\quad (3x - 15) \times x = 45000$

বা, $3x^2 - 15x = 45000$

বা, $x^2 - 5x = 15000$ [উভয়পক্ষকে $3$ দ্বারা ভাগ করে]

বা, $x^2 - 5x - 15000 = 0$

বা, $x^2 - 125x + 120x - 15000 = 0$

বা, $x(x - 125) + 120(x - 125) = 0$

বা, $(x - 125)(x + 120) = 0$

সুতরাং, $(x - 125) = 0$ অথবা $(x + 120) = 0$

বা, $x = 125$ বা, $x = -120$

যেহেতু সদস্য সংখ্যা ঋণাত্মক হতে পারে না, তাই $x$ এর মান $-120$ গ্রহণযোগ্য নয়।

সুতরাং, সমিতির সদস্য সংখ্যা $125$

### উদাহরণ ৩৫.

রফিক একটি কাজ $10$ দিনে করতে পারে। শফিক ঐ কাজ $15$ দিনে করতে পারে। তারা একত্রে কত দিনে কাজটি শেষ করতে পারবে?

**সমাধান:** মনে করি, তারা একত্রে $d$ দিনে কাজটি শেষ করতে পারবে।

| নাম | কাজ সম্পন্ন করার দিন | ১ দিনে কাজের সম্পন্ন অংশ | $d$ দিনে কাজের সম্পন্ন অংশ |

### উদাহরণ ৩৬.

একজন মাঝি স্রোতের প্রতিকূলে $t_1$ ঘণ্টায় $x$ কি.মি. যেতে পারে। স্রোতের অনুকূলে ঐ পথ যেতে তার $t_2$ ঘণ্টা লাগে। স্রোতের বেগ ও নৌকার বেগ কত?

**সমাধান:** ধরি, স্রোতের বেগ ঘণ্টায় $v$ কি.মি. এবং স্থির পানিতে নৌকার বেগ ঘণ্টায় $u$ কি.মি.।

তাহলে, স্রোতের অনুকূলে নৌকার কার্যকরী বেগ ঘণ্টায় $(u + v)$ কি.মি. এবং স্রোতের প্রতিকূলে নৌকার কার্যকরী বেগ ঘণ্টায় $(u - v)$ কি.মি.।

আমরা জানি, বেগ $=$ $\dfrac{\text{অতিক্রান্ত দূরত্ব}}{\text{সময়}}$

প্রশ্নানুসারে, $u + v = \dfrac{x}{t_2} \cdots \cdots (1)$

এবং $u - v = \dfrac{x}{t_1} \cdots \cdots (2)$

সমীকরণ $(1)$ ও $(2)$ যোগ করে পাই,

$2u = \dfrac{x}{t_2} + \dfrac{x}{t_1} = x\left(\dfrac{1}{t_1} + \dfrac{1}{t_2}\right)$ বা, $u = \dfrac{x}{2}\left(\dfrac{1}{t_1} + \dfrac{1}{t_2}\right)$

সমীকরণ $(1)$ ও $(2)$ বিয়োগ করে পাই,

$2v = \dfrac{x}{t_2} - \dfrac{x}{t_1} = x\left(\dfrac{1}{t_2} - \dfrac{1}{t_1}\right)$ বা, $v = \dfrac{x}{2}\left(\dfrac{1}{t_2} - \dfrac{1}{t_1}\right)$

সুতরাং, স্রোতের বেগ ঘণ্টায় $\dfrac{x}{2}\left(\dfrac{1}{t_2} - \dfrac{1}{t_1}\right)$ কি.মি. এবং নৌকার বেগ ঘণ্টায় $\dfrac{x}{2}\left(\dfrac{1}{t_1} + \dfrac{1}{t_2}\right)$ কি.মি.।

### উদাহরণ ৩৭.

একটি নল $12$ মিনিটে একটি খালি চৌবাচ্চা পূর্ণ করতে পারে। অপর একটি নল প্রতি মিনিটে $14$ লিটার পানি বের করে দেয়। চৌবাচ্চাটি খালি থাকা অবস্থায় দুইটি নল একসাথে খুলে দেওয়া হলে চৌবাচ্চাটি $96$ মিনিটে পূর্ণ হয়। চৌবাচ্চাটিতে কত লিটার পানি ধরে?

**সমাধান:** মনে করি, প্রথম নল দ্বারা প্রতি মিনিটে $x$ লিটার পানি প্রবেশ করে এবং চৌবাচ্চাটিতে মোট $y$ লিটার পানি ধরে।

প্রশ্নানুসারে, প্রথম নল দ্বারা $12$ মিনিটে চৌবাচ্চাটি পূর্ণ হয়

$\therefore y = 12x \cdots \cdots (1)$

আবার, দুইটি নল দ্বারা $96$ মিনিটে চৌবাচ্চাটি পূর্ণ হয়

$\therefore y = 96x - 96 \times 14 \cdots \cdots (2)$

সমীকরণ $(1)$ থেকে পাই, $x = \dfrac{y}{12}$

$x$ এর মান সমীকরণ $(2)$ এ বসিয়ে পাই,

$y = 96 \times \dfrac{y}{12} - 96 \times 14$

বা, $y = 8y - 96 \times 14$

বা, $7y = 96 \times 14$

বা, $y = \dfrac{96 \times 14}{7} = 192$

সুতরাং, চৌবাচ্চাটিতে মোট $192$ লিটার পানি ধরে।

### উদাহরণ ৩৮.

একটি বইয়ের মূল্য $24$ টাকা। এই মূল্য বই তৈরির ব্যয়ের $80\%$। বাকি মূল্য সরকার ভর্তুকি দিয়ে থাকেন। সরকার প্রতি বইয়ে কত টাকা ভর্তুকি দেন?

**সমাধান:** বাজার মূল্য = বই তৈরির ব্যয়ের $80\%$

আমরা জানি, $p = br$

এখানে, $p = 24$ টাকা এবং $r = 80\% = \dfrac{80}{100}$

$\therefore 24 = b \times \dfrac{80}{100}$

বা, $b = \dfrac{24 \times 100}{80}$

$\therefore b = 30$ টাকা

সুতরাং বই তৈরির ব্যয় $30$ টাকা

$\therefore$ ভর্তুকি $= (30 - 24)$ টাকা $= 6$ টাকা

সুতরাং সরকার প্রতি বইয়ে $6$ টাকা ভর্তুকি দেন।

### উদাহরণ ৩৯.

$r$ টাকায় $n$ সংখ্যক কমলা বিক্রয় করায় $r\%$ ক্ষতি হয়। $s\%$ লাভ করতে হলে, $1$ টাকায় কয়টি কমলা বিক্রয় করতে হবে?

**সমাধান:** ক্রয়মূল্য $100$ টাকা হলে, $r\%$ ক্ষতিতে বিক্রয়মূল্য $(100 - r)$ টাকা।

তাহলে, যখন বিক্রয়মূল্য $(100 - r)$ টাকা, তখন ক্রয়মূল্য $100$ টাকা।

$\therefore$ যখন বিক্রয়মূল্য $1$ টাকা, তখন ক্রয়মূল্য $\dfrac{100}{100 - r}$ টাকা।

আবার, ক্রয়মূল্য $100$ টাকা হলে, $s\%$ লাভে বিক্রয়মূল্য $(100 + s)$ টাকা।

$\therefore$ ক্রয়মূল্য $100$ টাকা হলে, $s\%$ লাভে বিক্রয়মূল্য $\left(\dfrac{100 + s}{100} \times \dfrac{100}{100 - r}\right)$ টাকা

$= \dfrac{100 + s}{100 - r}$ টাকা।

সুতরাং, $\dfrac{100 + s}{100 - r}$ টাকায় বিক্রয় করতে হবে $n$ সংখ্যক কমলা

$\therefore 1$ টাকায় বিক্রয় করতে হবে $n \times \left(\dfrac{100 - r}{100 + s}\right)$ সংখ্যক কমলা

সুতরাং, $\dfrac{n(100 - r)}{100 + s}$ সংখ্যক কমলা বিক্রয় করতে হবে।

### উদাহরণ ৪০.

শতকরা বার্ষিক $7$ টাকা হার সরল মুনাফায় $650$ টাকার $6$ বছরের মুনাফা কত?

**সমাধান:** আমরা জানি, $I = Pnr$

এখানে, $P = 650$ টাকা, $n = 6$ বছর, শতকরা মুনাফার হার $s = 7$ টাকা

$\therefore r = \dfrac{s}{100} = \dfrac{7}{100}$

$\therefore I = 650 \times 6 \times \dfrac{7}{100} = 273$

সুতরাং, মুনাফা $273$ টাকা।

### উদাহরণ ৪১.

বার্ষিক শতকরা $6$ টাকা হার চক্রবৃদ্ধি মুনাফায় $15000$ টাকার $3$ বছরের সবৃদ্ধিমূল ও চক্রবৃদ্ধি মুনাফা নির্ণয় করো।

**সমাধান:** আমরা জানি, $C = P(1 + r)^n$ [যেখানে $C$ চক্রবৃদ্ধির ক্ষেত্রে সবৃদ্ধিমূল]

দেওয়া আছে, $P = 15000$ টাকা, $r = 6\% = \dfrac{6}{100}, n = 3$ বছর

$\therefore C = 15000\left(1 + \dfrac{6}{100}\right)^3 = 15000\left(1 + \dfrac{3}{50}\right)^3 = 15000\left(\dfrac{53}{50}\right)^3$

$\qquad = 15000 \times \dfrac{53}{50} \times \dfrac{53}{50} \times \dfrac{53}{50} = \dfrac{446631}{25} = 17865.24$

$\therefore$ সবৃদ্ধিমূল $= 17865.24$ টাকা

$\therefore$ চক্রবৃদ্ধি মুনাফা $= (17865.24 - 15000)$ টাকা $= 2865.24$ টাকা।

### উদাহরণ ৪২.

টাকায় $10$টি আইসক্রিম এর কাঠি বিক্রয় করলে $x\%$ ক্ষতি হয়। টাকায় কয়টি বিক্রয় করলে $z\%$ লাভ হবে?

**সমাধান:** ক্রয়মূল্য $100$ টাকা হলে $x\%$ ক্ষতিতে বিক্রয়মূল্য $= (100 - x)$ টাকা

বিক্রয়মূল্য $(100 - x)$ টাকা হলে ক্রয়মূল্য $100$ টাকা

$\therefore$ বিক্রয়মূল্য $1$ টাকা হলে ক্রয়মূল্য $\dfrac{100}{100 - x}$ টাকা

অর্থাৎ, $10$টি আইসক্রিম কাঠির ক্রয়মূল্য $\dfrac{100}{100 - x}$ টাকা

$\therefore 1$টি আইসক্রিম কাঠির ক্রয়মূল্য $\dfrac{100}{(100 - x) \times 10}$ টাকা

আবার ক্রয়মূল্য $100$ টাকা হলে $z\%$ লাভে বিক্রয়মূল্য $(100 + z)$ টাকা

ক্রয়মূল্য $100$ টাকা হলে বিক্রয়মূল্য $(100 + z)$ টাকা

ক্রয়মূল্য $1$ টাকা হলে বিক্রয়মূল্য $\dfrac{100 + z}{100}$ টাকা

$\therefore$ ক্রয়মূল্য $\dfrac{100}{(100 - x) \times 10}$ টাকা হলে

বিক্রয়মূল্য $\dfrac{100 + z}{100} \times \dfrac{100}{(100 - x) \times 10}$ টাকা $= \dfrac{(100 + z)}{(100 - x) \times 10}$ টাকা

$1$টি আইসক্রিম কাঠির বিক্রয়মূল্য $\dfrac{(100 + z)}{(100 - x) \times 10} = \dfrac{100 + z}{1000 - 10x}$ টাকা

অর্থাৎ, টাকায় $\dfrac{1000 - 10x}{100 + z}$ টি আইসক্রিম কাঠি বিক্রয় করতে হবে।
